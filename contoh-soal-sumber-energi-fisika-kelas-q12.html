skala = 1 : 300.000

A - P = 14 cm x 300.000

= 4.200.000 cm = 42 km

P - B = 16 cm x 300.000

= 4.800.000 cm = 48 km

A - P - B = 42 km + 48 km = 90 km

A - Q = 12 cm x 300.000

= 3.600.000 cm = 36 km

Q - B = 21 cm x 300.000

= 6.100.000 cm = 61 km

A - Q - B = 36 km + 61 km = 97 km

Selisih dari kedua jalur tersebut adalah

97 km - 90 km = 7 km

15. 9. Sebuah perusahaan yang memproduksi meja kursi, memproduksi dan menjual produknya masing-masing dengan harga (dalam ribuan rupiah) 520 per unit meja dan 230 per unit kursi. Biaya produksi mingguan untuk menghasilkan dua jenis produk dimaksud mengikuti persamaan (dalam ribuan rupiah), TC = Q12 + Q22 + 2 Q1 Q2 + 30 Q1 + 40 Q2 + 100 Dengan: Q1 kuantita meja yang diproduksi, dan Q2 kuantita kursi yang diproduksi. Sementara itu, berdasar pengalaman hasil penjualan sebelumnya, jumlah kursi yang terjual adalah 4 (empat) kali dari jumlah meja. Tentukan berapa meja dan kursi harus diproduksi perusahaan tiap minggunya agar keuntungan menjadi maksimum.


Misalkan,
x banyaknya meja yang terjual,
y banyaknya kursi yang terjual.

Harga meja per unit = 520 ribu rupiah
Harga kursi per unit = 230 ribu rupiah

Biaya produksi mengikuti persamaan,
TC = x² + y² + 2xy + 30x + 40 y + 100 ribu rupiah

"jumlah kursi yang terjual adalah 4 (empat) kali dari jumlah meja"
y = 4x

Maksimumkan keuntungan,
Keuntungan = Total pendapatan - total biaya
K(x, y) = 520x + 230y - (x² + y² + 2xy + 30x + 40 y + 100)
K(x, y) = 520x + 230y - x² - y² - 2xy - 30x - 40 y - 100
K(x, y) = -x² - y² - 2xy + 490x + 190y - 100

Substitusi, y = 4x
K(x) = -x² - (4x)² - 2x(4x) + 490x + 190(4x) - 100
K(x) = -x² - 16x² - 8x² + 490x +760x - 100
K(x) = -25x² + 1250x - 100

Nilai maksimum K(x) berada pada titik stasionernya yaitu
K'(x) = 0
-50x + 1250 = 0
-50x = -1250
x = 25

Diperoleh nilai x = 25,
y = 4(25) = 100

Jadi keuntungan perusahaan tersebut maksimum ketika diproduksi 25 meja dan 100 kursi dengan keuntungan 15525 ribu rupiah.

17. Mohon bantuannya : Optimum Utilitas kendala anggaran: Diketahui fungsi tujuan :U = Q12 +2Q22 +5Q1.Q2. Dengan kendala:5Q1 + 10Q2 = 90. Tentukan: a. Jumlah Q1 dan Q2 yang memaksimum Utilitas; b. Tentukan U optimum; c. Buktikan bahwa U Optimum Maksimum.


Jawaban:

U = Q₁² + 2Q₂² + 5Q₁Q₂

Fungsi Kendala: g : 5Q₁ + 10Q₂ = 90

a. Jumlah Q₁ dan Q₂ yang memaksimumkan utilitas

Fungsi Lagrange:

L = Q₁² + 2Q₂² + 5Q₁Q₂ + λ (90 - 5Q₁ - 10Q₂)

ΔL / ΔQ₁ = 2Q₁ + 5Q₂ - 5λ = 0

ΔL / ΔQ₂ = 4Q₂ + 5Q₁ - 10λ = 0

ΔL / Δλ = 90 - 5Q₁ - 10Q₂ = 0

2Q₁ + 5Q₂ - 5λ = 0

(2Q₁ + 5Q₂) / 5 = λ ..........Persamaan 1

4Q₂ + 5Q₁ - 10λ = 0

(4Q₂ + 5Q₁) / 10 = λ ..........Persamaan 2

Gabungkan persamaan 1 dan 2

λ = λ

(2Q₁ + 5Q₂) / 5 = (4Q₂ + 5Q₁) / 10

20Q₁ + 50Q₂ = 20Q₂ + 25Q₁

50Q₂ - 20Q₂ = 25Q₁ - 20Q₁

30Q₂ = 5Q₁

Q₁ = 6Q₂

90 - 5Q₁ - 10Q₂ = 0

90 - 5 (6Q₂) - 10Q₂ = 0

90 - 30Q₂ - 10Q₂ = 0

90 = 40Q₂

Q₂ = 9/4

Q₁ = 6Q₂

Q₁ = 6 (9/4)

Q₁ = 27/2

b. Tingkat U optimum

U = Q₁² + 2Q₂² + 5Q₁Q₂

U = (27/2)² + 2 (9/4)² + 5 (27/2) (9/4)

U = 344.25

c. Buktikan bahwa U optimum

Menggunakan Bordered Hessian Matrix:

g₁ = Δg/ΔQ₁ = 5

g₂ = Δg/ΔQ₂ = 10

L₁₂ = L₂₁ = Δ(ΔL/ΔQ₁) / ΔQ₂ = 5

L₁₁ = Δ(ΔL/ΔQ₁) / ΔQ₁ = 2

L₂₂ = Δ(ΔL/ΔQ₂) / ΔQ₂ = 4

Bordered Hessian:

|  0   g₁   g₂   |

|  g₁  L₁₁   L₁₂  |

|  g₂ L₂₁   L₂₂ |

|  0   5   10   |

|  5   2   5     |

|  10  5   4    |

determinant dari bordered hessian matrix = -5 (5x4 - 10 x 5) + 10 (5 x 5 - 10 x 2)

= -5 (20 - 50) + 10 (25 - 20)

= -5 (-30) + 10 (5)

= 150 + 50

= 200 > 0 (Maksimum)
