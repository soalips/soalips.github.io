25. Dari 50 siswa dalam sebuah kelas, terdapat 36 siswa gemar pljrn mtk 32 gemar pljrn fisika. Jika 9 siswa tidak gemar mata pelajaran mtk maupun fisika, bnyk siswa yg gemar mtk dan fisika adalah...a. 5 siswab. 9 siswac. 22 siswad. 27 siswa​


Penjelasan dengan langkah-langkah:

jumlah siswa = 50

gemar mtk = 36 - x

gemar fisika = 32 - x

tidak gemar keduanya = 9

gemar mtk dan fisika = x = ?

jawab

jumlah siswa = gemar mtk + gemar fisika + tidak gemar keduanya + gemar mtk dan fisika

50 = (36-x) + (32-x) + 9 + x

50 = 36 + 32 + 9 - x - x + x

50 = 77 + x

x = 50 - 77

x = -27

Jadi banyak siswa yg gemar mtk dan fisika adalah 27 siswa.

#sejutapohon

Langkah-langkah

n(s) = (36 - x) + (32 - x) + 9 + x

50 = 36 + 32 + 9 - x

50 = 77 - x

x = 77 - 50

x = 27

Jadi, banyak siswa yang gemar mtk dan fisika sebanyak 27 siswa.

39. Q. Mulailah pljrn SMP Soal terlampir Kaks!​


Jawab: A. -7

Penjelasan dengan langkah-langkah:

Dilampirkan

Jawaban:

~Fungsi

=============

Diketahui :

f(x) = ax + b

f(2) = 1

f(4) = 7

Ditanyakan :

a + 2b = .. ?

Jawab :

f(2) = a(2) + b = 1

2a + b = 1

f(4) = a(4) + b = 7

4a + b = 7

b = 7 - 4a

Substitusi persamaan (2) ke persamaan (1)

2a + b = 1

2a + ( 7 - 4a ) = 1

2a + 7 - 4a = 1

-2a + 7 = 1

-2a = 1 - 7

-2a = -6

a = -6/-2

a = 3

b = 7 - 4a

= 7 - 4(3)

= 7 - 12

= -5

a + 2b

» 3 + 2(-5)

» 3 + -10

» -7

Kesimpulan :

Jadi, a + 2b hasilnya adalah -7 (A)

============================
