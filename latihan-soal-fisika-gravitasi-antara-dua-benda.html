3. Dua benda A dan B masing-masing 36 kg dan 4 kg. Keduanya terpisah sejauh 80 cm. Jika benda C terletak diantara benda A dan B. Tentukan letak benda C sehingga tidak mengalami gaya gravitasi !!!Bantu jawab kk... soal fisika Kelas : 10Bab : Medan Gravitasi ​


Diket

massa benda A = 36kg

massa benda B = 4kg

jarak benda A ke benda B = 80cm.

Tanya

Letak benda C agar tidak mengalami gaya grafitasi.

Jawab

C terletak antara A dan B

misalkan

jarak C ke A = n

jarak C ke B = 80 - n

Agar grafitasi = 0 maka

Fca = Fcb

ma / n² = mb / (80 - n)²

36 / n² = 4 / (80 - n)²

6²/n² = 2² / (80 - n)²

6 / n = 2 / 80 - n

2n = 480 - 6n

2n + 6n = 480

8n = 480

n = 60

Jadi Benda C diletakkan dengan jarak

60cm dari Benda A, 20cm dari Benda B.

16. Gaya gravitasi antara dua buah bebda yang massanya masing masing m1 dan m2 dan terpisah pada jarak r adalah F. Jika jarak antara kedua benda dijadikan 2r, gaya gravitasi diantara kedua benda menjadi


Materi : Gaya Gravitasi
Sub bab : Gaya Gravitasi

Diketahui :
F1 = F
r1 = r
r2 = 2r

Ditanya :
F2 = ?

Penyelesaian :
F = G. m1m2 / r²
Fr² = G. m1m2

F1r1² = F2r2²
Fr² = F2(2r)²
Fr² = F2.4r²
F = 4F2
F2 = 1/4 F

Jadi, gaya gravitasi kedua benda menjadi ¼F

25. soal fisika latihan usbn mohon dibantu


RELATIVITAS

Soal nomor (1)

Diketahui:

Kecepatan pesawat A menurut pengamat di bumi, vap = 0,5c

Kecepatan pesawat B menurut pengamat di bumi, vbp = -0,4c (arah berlawanan dengan pesawat A)

Ditanya:

a. Kecepatan pesawat A menurut pilot pesawat B, vab = ...?

b. Kecepatan pesawat B menurut pilot pesawat A, vba = ...?

Pembahasan:

a. Kecepatan pesawat A menurut pilot pesawat B

vab = (vap + vpb)/(1+vap.vpb/c²)

vab = (0,5c + 0,4c)/(1+0,5c.0,4c/c²)

vab = 0,9c/1,2

vab = 3c/4

b. Kecepatan pesawat B menurut pilot pesawat A

vba = (vbp + vpa)/(1+vbp.vpa/c²)

vab = [-0,4c + (-0,5c)]/[1+(-0,4c).(-0,5c)/c²]

vab = -0,9c/1,2

vab = -3c/4

Soal nomor (2)

Diketahui:

Usia awal A, ta1 = 10 tahun

Usia awal B, tb1 = 10 tahun

Usia akhir A, ta2 = 40 tahun

Ditanya:

Usia B menurut A, tb' = ...?

Pembahasan

Selisih usia B menurut pengamat di bumi

Δtb = 40 - 10 = 30 tahun

Selisih usia B menurut A

Δtb' = Δtb/√(1 - v²/c²)

Δtb' = 30/√[1 - (0,8c)²/c²]

Δtb' = 30/√(1 - 0,64)

Δtb' = 30/√0,36

Δtb' = 30/0,6

Δtb' = 50

tb' = 10 + 50 = 60 tahun

Jadi usia B menurut A adalah 60 tahun

Soal nomor (3)

Diketahui:

Kecepatan pesawat 1 menurut pengamat di bumi, v13 = 0,5c

Kecepatan pesawat 2 menurut pengamat di bumi, v23 = -0,25c (arah berlawanan dengan pesawat 1)

Ditanya:

a. Kecepatan pesawat 1 menurut pilot pesawat 2, v12 = ...?

b. Kecepatan pesawat 2 menurut pilot pesawat 1, v21 = ...?

Pembahasan:

a. Kecepatan pesawat 1 menurut pilot pesawat 2

v12 = (v13 + v32)/(1+v13.v32/c²)

v12 = (0,5c + 0,25c)/(1+0,5c.0,25c/c²)

v12 = 0,75c/1,125

v12 = c/15

b. Kecepatan pesawat 2 menurut pilot pesawat 1

v21 = (v23 + v31)/(1+v23.v31/c²)

v21 = [-0,25c + (-0,5c)]/[1+(-0,25c).(-0,5c)/c²]

v21 = -0,75c/1,125

v21 = -c/15

Soal nomor (4)

Diketahui:

Panjang batang diamati saat diam, Lo = 1 m

Panjang batang diamati saat bergerak, L = 0,8 m

Ditanya:

Hitung kecepatan gerak batang, v = ...?

Pembahasan:

L = Lo.√(1 - v²/c²)

0,8 = 1.√(1 - v²/c²)

0,8 = √(1 - v²/c²)

0,64 = 1 - v²/c²

v²/c² = 1 - 0,64

v²/c² = 0,36

v² = 0,36c²

v = 0,6c
