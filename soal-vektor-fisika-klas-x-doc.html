30. Fisika klas X.Pakai penjelasanya yah ,Gomawo​


DiketahuiMassa balok A ([tex]m_a[/tex]) = 1,2 kgMassa balok B ([tex]m_b[/tex]) = 1,8 kgPercepatan gravitasi (g) = 10 m/s²

DitanyaBesar percepatan system = ..?

Pembahasan

Dalam hal ini massa katrol diabaikan [tex](m_k = 0)[/tex]

Dalam gambar terlihat balok B menarik balok A dan sebaliknya. Karena massa Balok B lebih besar dari massa Balok A [tex](m_b > m_a)[/tex]. Sehingga akan diperoleh:

[tex]\sum F = \sum m ~.~a[/tex]

[tex]w_b - w_a = (m_a + m_b)~.~a[/tex]

[tex]a = \frac{w_b - w_a}{m_a + m_b}[/tex]

[tex]a = \frac{m_b.g ~-~ m_a.g}{m_a + m_b}[/tex]

[tex]a = g\times \frac{m_b - m_a}{m_a + m_b}[/tex]

Maka, Percepatannya adalah

[tex]a = g\times \frac{1,8~kg~-~ 1.2~kg}{1,8~kg ~+~ 1,2~kg}[/tex]

[tex]a = 10~m/s^2 \times \frac{0,6~kg}{3~kg}[/tex]

[tex]a = 10~m/s^2 \times 0,2[/tex]

[tex]a = \boxed{2 ~m/s^2}[/tex]

→ Jadi, percepatannya adalah 2 m/s²

34. Hitung nilai x dan besar sudut DOC ​


∠COD  = 54°

PembahasanDiketahui :

Perhatikan gambar!

∠AOC = 90°

∠AOC = 36°

Ditanyakan :

∠DOC = 90°

Jawab :

Pada gambar ∠AOB = 180°

maka ∠AOC + ∠COD + ∠DOB  = 180°

⇔ ∠AOC + ∠COD + ∠DOB  = 180°

⇔    90°   + ∠COD +   36 °   = 180°

⇔              ∠COD  +   126 °   = 180°

⇔                              ∠COD  = 180° - 126°

⇔                              ∠COD  = 54°

∴ Jadi besar ∠COD  = 54°
