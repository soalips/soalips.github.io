32. Tolong bantu jawab iyah n0 34 ??


Jawaban:

~ Math

Dik :

panjang = 20 cm

lebar = 10 cm

tinggi = 5 cm

Dit :

Luas permukaan (Lp) = ?

Penyelesaian :

Lp = 2pl + 2pt + 2lt

Lp = 2(20)(10) + 2(20)(5) + 2(10)(5)

Lp = 400 + 200 + 100

Lp = 700 cm²

Jadi , luas permukaan balok tersebut adalah 700 cm²

===

Mapel : Matematika

Kelas : 8

Materi : Bangun Ruang

Kode : 8.2.8

Jawab:

700 cm²

Penjelasan dengan langkah-langkah:

Diketahui :

Panjang balok = 20 cm

Tinggi balok = 5 cm

Lebar balok = 10 cm

Ditanya :

Luas Permukaan balok .... ?

Jawab :

Rumus untuk menentukan luas permukaan balok adalah

LP Balok = 2 x (p x l) + 2 x (p x t) + 2 x (l x t)

LP Balok = 2 x (20 x 10) + 2 x (20 x 5) + 2 x (10 x 5)

LP Balok = 2 x 200 + 2 x 100 + 2 x 50

LP Balok = 400 + 200 + 100

LP Balok = 600 + 100

LP Balok = 700 cm²

Jadi, Luas permukaan balok tersebut adalah 700 cm²
