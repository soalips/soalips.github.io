29. tebak-tebakan matematika​


Jawaban:

A = 77

A + B = 86

A - B = 68

________ +

2A = 154

A = 154 / 2

A = 77

A + B = 86

77 + B = 86

B = 86 - 77

B = 9

A + B = 86

A - B = 68

Eliminasi B:

A + B = 86

A - B = 68

------------- +

2A = 154

A = 77

B = 9

Nilai A adalah 77
