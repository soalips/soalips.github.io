27. bantu nih persiapan sbm


[tex] \sqrt{x+5} \ \textless \ \sqrt{2x+7} \\ [/tex]
Ingat, syarat akar harus lebih dari sama dengan 0
Maka
[tex]x+5 \geq 0 \\ x \geq -5 \\ 2x+7 \geq 0 \\ x \geq -\frac{7}{2} [/tex]
Kemudian kuadratkan kedua sisi, sehingga :
[tex]x+5 \ \textless \ 2x+7 \\ x \ \textgreater \ -2[/tex]

Kemudian iriskan himpunan penyelesaiannya, lebih mudah menggunakan istilah selang, maka akan dihasilkan Hp : x > -2
x + 5 < 2x + 7 (akar ketemu akar jadi hilang)
x - 2x < 7 - 5
-x < 2
x > 2/-1
x > -2 (yang c)
