MIsalkan:

u = sin 3x
⇔ du/ dx = 3 cos 3x

⇔ dx = du/ 3 cos 3x

sehingga,

∫ sin³ 3x cos 3x dx

= ∫ U³ . cos 3x . (du/ 3cos 3x)

= 1/3 ∫ U³ du

= 1/3 . 1/4 . U^4 + C

= (1/12) sin^4 3x + C

Jadi, ∫ sin³ 3x cos 3x dx = (1/12) sin^4 3x + C

7. Himpunan penyelesaian dari persamaan cos 2x – 3 cos x + 2 = 0, 0˚ ≤ x ≤ 360˚ adalah … (Ujian Nasional Matematika Tahun 2011) **Super Intensif UN (Ujian Nasional) SMA/MA IPA , Penerbit: Bisa!Publishing 2016, Penulis: Muhammad Amien & Team


Persamaan Trigonometri

cos 2x - 3 cos x + 2 = 0

⇔ (2 cos² x - 1) - 3 cos x + 2 = 0

⇔ 2 cos² x - 3 cos x + 1 = 0

⇔ (2 cos x - 1)(cos x - 1) = 0

⇔ cos x =1/2 atau cos x = 1

cos x = 1/2 → x = 60, 300

cos x = 1 → x = 0, 360

Jadi, Himpunan penyelesaian persamaan tersebut adalah {0°, 60°, 300°, 360°}

19. Nilai lim x → 0 dari (1 – cos 2x)/ (1 – cos 4x) = … (Ujian Nasional Matematika Tahun 2011) **Super Intensif UN (Ujian Nasional) SMA/MA IPA , Penerbit: Bisa!Publishing 2016, Penulis: Muhammad Amien & Team


Limit Trigonometri

lim x→0 dari (1 - cos 2x)/ (1 - cos 4x)

⇔ lim x→0 (2 sin² x)/ (2 sin² 2x)

⇔ 2 . x²/ 2 . (2x)²

⇔ 1/4

Jadi, nilai dari lim x→0 untuk persamaan trigonometri tersebut adalah 1/4.

PEMBAHASAN

Logaritma adalah invers atau kebalikan dari suatu perpangkatan.

Pada aⁿ = b maka bentuk logaritmanya adalah ᵃlog b = n. Pada penulisan logaritma ᵃlog b = n, a disebut bilangan pokok, b disebut bilangan numerus atau bilangan yang dicari nilai logaritmanya (b > 0), dan dan c adalah hasil logaritma. Jika nilai a sama dengan 10, biasanya 10 tidak dituliskan sehingga menjadi log b = c.

Sifat-sifat logaritma :

a. ᵃlog a = 1

b. ᵃlog aⁿ = n

c. ᵃlog bⁿ = n. ᵃlog b

d.  ᵃlog (b.c) = ᵃlog b + ᵃlog c

e. ᵃlog b x ᵇlog c = ᵃlog c

f.  ᵃlog (ᵇ/ₓ) = ᵃlog b - ᵃlog x

[tex]g.^{a}log~b=\frac{1}{^{b}log~a}[/tex]

h. ᵃlog (b/c) = - ᵃlog (c/b)
