2. tolong dijawab soal fisika tentang kapasitor ini


Pada Rangkaian Capasitansi.

Ketika beberapa capasitor di rangkai secara seri.
1/Cs = 1/C + 1/C +...
1/Cs = n 1/C
Cs = C/n

Ketika n capasitor dirangkai secara paralel.
Cp = C + C +...
Cp = nC

Kemudian rangkaian seri dan paralel di rangkai secara seri

1/C = 1/Cp + 1/Cs

1/C = 1/nC + n/C
1/C = (1/nC + n^2/nC)
1/C = (1+ n^2) / nC

C = nC / (1 + n^2)

Jadi, capasitor penggantinya adalah nC/(1+n^2)

maaf kalau salah

[tex]\boxed{\text{Subsoal A}}[/tex]

Kapasitansi kapasitor ekuivalen:

[tex]\displaystyle \frac{1}{C_e} = \frac{1}{C_1} + \frac{1}{C_2}\\\frac{1}{C_e} = \frac{2}{10^{-11}~F}\\C_e = \frac{1}{2}\times10^{-11} ~F\\\therefore~~ C_e = \boxed{0.5\times10^{-11}~F}[/tex]

[tex]\boxed{\text{Subsoal B}}[/tex]

Muatan total dalam rangkaian kapasitor:

[tex]\displaystyle Q_{tot} = C_eV\\Q_{tot}=(0.5\times10^{-11~}F)(12~V)\\\therefore~~Q_{tot}=\boxed{6\times10^{-11}~C}[/tex]

[tex]\boxed{\text{Subsoal C}}[/tex]

Muatan pada masing-masing kapasitor:

[tex]\displaystyle Q= \frac{1}{2}Q_{tot}\\Q= \frac{1}{2}(6\times10^{-11}C)\\\therefore~~Q=\boxed{3\times10^{-11}C}[/tex]

[tex]\boxed{\text{Subsoal D}}[/tex]

Tegangan jepit pada masing-masing kapasitor:

[tex]\displaystyle V_{jepit} = \frac{Q}{C}\\\\V_{jepit}=\frac{3\times10^{-11}C}{10^{-11}F}\\\\\therefore~~V_{jepit} = \boxed{3~V}[/tex]

25. Contoh soal kapasitansi kapasitor


Jawab:

Sebuah kapasitor dipasang pada tegangan 220 volt. Bila muatan yang tersimpan 110 μC, tentukan kapasitas kapasitornya!

Penjelasan dengan langkah-langkah:

Diketahui:

V = 220 volt

Q = 110 μC

Ditanya: C =…?

Pembahasan:

Gunakan persamaan berikut untuk mencari kapasitas kapasitor.

[tex]C=\frac{Q}{V} \\\ \\C=\frac{110}{220} \\\\= 0,5uf[/tex]

Jadi, kapasitas kapasitornya adalah 0,5 μF.

• Kapasitas listrik

C₁ = K₁ εₒ A / d
C₁ = 1 • εₒ A / d
C₁ = εₒ A / d

diberi potensial V₁ = 70 V, sehingga memiliki muatan listrik Q dan memenuhi persamaan
C₁ = Q / V₁
→ Q = C₁ V₁

Kapasitor kemudian dilepas dari sumber tegangan dan disisipkan dielektrik dengan K₂ = 3,5.
Kapasitas listrik kapasitor menjadi
C₂ = K₂ εₒ A / d
C₂ = 3,5 • εₒ A / d
C₂ = 3,5 C₁

Muatan listrik yang tersimpan pada kapasitor tetap besarnya, sehingga tegangan turun menjadi V₂ dan memenuhi persamaan
C₂ = Q / V₂
→ Q = C₂ V₂

Hubungan potensial listrik sebelum dan sesudah dilepas dari sumber tegangan
Q = Q
C₁ V₁ = C₂ V₂
V₂ = C₁ V₁ / C₂
V₂ = C₁ • 70 / (3,5 C₁)
V₂ = 70 / 3,5
V₂ = 20 V ✔️

40. Sebuah kapasitor 16 µF dihubungkan dengan sumber tegangan 16 V, sedangkan kapasitor 12 µF dihubungkan dengan tegangan 32 V Jika kedua kapasitor digabungkan, hitunglah muatan akhir kapasitor masing-masing.​


Muatan kapasitor masing-masing jika sebuah kapasitor 16 µF dihubungkan dengan sumber tegangan 16 V, sedangkan kapasitor 12 µF dihubungkan dengan tegangan 32 V dan jika kedua kapasitor digabungkan adalah:

1. Muatan 1 = 2,56 x 10⁻⁴ Coulomb

2. Muatan 2 = 3,84 x 10⁻⁴ Coulomb

Penjelasan dengan langkah-langkahDikatahui: Sebuah kapasitor 16 µF dihubungkan dengan sumber tegangan 16 V, sedangkan kapasitor 12 µF dihubungkan dengan tegangan 32 V Ditanya: Jika kedua kapasitor digabungkan, hitunglah muatan akhir kapasitor masing-masing.​!Jawab:

Langkah 1

Untuk menentukan muatan pada masing-masing kapasitor, gunakan persamaan:

Q = C x V

Dengan:

Q = muatan listrik pada masing-masing keping

C = kapasitas kapasitor (F)

V = beda potenisal (volt)

Langkah 2

Maka,

Muatan 1

Q₁ = C₁ x V₁

= (16 x 10⁻⁶ F) x (16 V)

= 2,56 x 10⁻⁴ Coulomb

Muatan 2

Q₂ = C₂ x V₂

= (12 x 10⁻⁶ F) x (32 V)

= 3,84 x 10⁻⁴ Coulomb
