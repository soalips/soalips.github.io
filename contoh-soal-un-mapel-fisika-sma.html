15. 1 soal mapel fisika kelas 12​


Penjelasan:

HAMBATAN LISTRIK.

Diketahui:

d = 0,2 mm => r = 0,1 mm

R = 5 ohm

rho = 3,5 × 10^-5 ohm. m

Ditanya:

panjang batang karbon, L = ?

Jawab:

luas penampang karbon =

A = πr² = 3,14 × (0,1)² = 3,14 × 0,01

= 3,14 × 10`² mm² = 3,14 × 10^-8 m²

R = rho. L/A

L = A. R/rho

= 3,14 × 10^-8 × 5/ 3,5 × 10^-5

= 15,7 × 10^-8 / 3,5 ×10^-5

= 4,5 × 10`³ m

= 4,5 mm

25. Quizzz [Mapel: UN]1. Permutasi dari: "Quizz"2. Kombinasi dari: "Quizz"3. NaCl terdiri dari...Ini sih bukn soal UN, cuman campur mapelnya jadi kasih UN​

n!/r! 5!/2!5 x 4 x 3 x 2 x 1/2 x 1120/260 susunan

Quizz = 5 huruf, 2 unsur ganda

n!/r! (n - r)!5!/2! (5 - 2)!5 x 4 x 3 x 2 x 1/2 x 1 x 3! 120/2 × 6120/1210 cara

NaCl terdiri dari Natrium Klorida.

Na = NatriumCl = Klorida

Jawab:

1. QUIZZ

Total huruf : 5

Unsur ganda : 2

p = [tex]\frac{n!}{k!}[/tex]

p = [tex]\frac{5\times4\times3\times 2\times 1}{2\times 1}[/tex]

p = [tex]\frac{120}{2}[/tex]

p = 60 Susunan

2. QUIZZ

Total huruf : 5

Unsur ganda : 2

C = [tex]\frac{n!}{(n-r)!\times k!}[/tex]

C = [tex]\frac{5!}{(5-2)!\times2!}[/tex]

C = [tex]\frac{5!}{3!\times2!}[/tex]

C = [tex]\frac{5\times4\times3\times2\times1}{(3\times2\times1\times2\times1)}[/tex]

C = [tex]\frac{120}{12}[/tex]

C = 10 Susunan

3. NaCl terdiri dari unsur penyusun Natrium dan Klorin.

Pembahasan

Metode-metode dalam kaidah pencacahan:

Kaidah perkalian. Prinsipnya mengalikan banyak kejadian yang mungkin dari tiap penyusunnya.Permutasi. Digunakan untuk menghitung banyak cara jika tidak ada pengulangan, dan dibulak-balik dianggap berbeda.Kombinasi. Digunakan untuk menghitung banyak cara jika tidak ada pengulangan, dan dibulak-balik dianggap sama.

Permutasi merupakan susunan atau urutan yang berbeda satu sama lain yang terbentuk dari sebagian atau dari seluruh objek.

38. ( BIG POINT )MAPEL : FISIKAKELAS : 10 SMAJAWAB BESERTA PENJELASANJANGAN NGASAL-@@-​


diket:

rA = 10 cm = 0,1 m

rB = 5 cm = 0,05 m

rC = 15 cm = 0,15 m

ωA = 2 rad/s

ditanya:

a. kelajuan linier roda A (vA)....?

b. kecepatan sudut roda B (ωB)...?

c. kelajuan linier roda B (vB)...?

d. kelajuan linier roda C (vC)...?

e. kecepatan sudut roda C (ωC)....?

jawab:

a. kelajuan linier roda A (vA)

   vA = rA ωA

   vA = 0,1 m x 2 rad/s

        = 0,2 m/s

Jadi, kelajuan linier roda A adalah 0,2 m/s.

b. kecepatan sudut roda B (ωB)

   karena roda A dan roda B saling menempel, maka

   ωA = ωB

   ωB = 2 rad/s

Jadi, kecepatan sudut roda B adalah 2 rad/s.

c. kelajuan linier roda B (vB)

   vB = rB ωB

   vB = 0,05 m x 2 rad/s

        = 0,1 m/s

Jadi, kelajuan linier roda B adalah 0,1 m/s.

d. kelajuan linier roda C (vC)

   karena roda B dan roda C terhubung rantai, maka

   vB = vC

   vC = 0,1 m/s

Jadi, kelajuan linier roda C adalah 0,1 m/s.

e. kecepatan sudut roda C (ωC)

   ωC = vC / rC

   ωC = 0,1 m/s : 0,15 m

          = 0,67 rad/s

Jadi, kecepatan sudut roda C adalah 0,67 rad/s.
