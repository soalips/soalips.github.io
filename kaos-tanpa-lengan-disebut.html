x = kaos lengan pendek

y = kaos lengan panjang

20.000x + 25.000y ≤ 1.800.000

4x + 5y ≤ 360

x + y ≤ 80

x ≥ 0

y ≥ 0

f(x) = 5.000x + 4.000y

Jawaban:

x + y = 80 --> x = 80 - y

25000x + 20000y = 1800000

model matematika hanya sampai batas ini

Ini penyelesaiannya

25000(80-y) + 20000y = 1800000

2000000 - 25000y + 20000y = 1800000

2000000-1800000 = 25000y - 20000y

200000 = 5000y

y = 40

x = 80 - 40 = 40

kalo ditanya keuntungan total:

40 x 4000 + 40 x 5000

= rp 360000
