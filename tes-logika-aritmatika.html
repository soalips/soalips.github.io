15 liter → 225 km

x = (15 . 180)/225

x = 2.700/225

23. Tes Deret Angka, Pilih satu jawaban yang sesuai dengan pola angka Aritmatika. 106, 105, 102, 97, 90, .....​


Materi : Barisan dan Geometri

Pola barisan bertingkat

106, 105, 102, 97, 90, ... [ a = 106 ]

-1, -3, -5, -7, ... [ b = -1 ]

-2, -2, -2, ... [ c = -2 ]

Maka rumus pola :

Un = a + b( n - 1 ) + c/2( n - 1 )( n - 2 )

Un = 106 - ( n - 1 ) - 2/2( n² - 3n + 2 )

Un = 106 - n + 1 - n² + 3n - 2

Un = 105 + 2n - n²

Semoga bisa membantu
