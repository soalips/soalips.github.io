7. Jika efisiensi transformator di atas 80%, maka nilai kuat arus keluaran dan daya keluarannya menjadi


Penjelasan:

Diketahui :

Efisiensi transformator, η = 80%

Tegangan primer (Input), Vp = 220 volt

Kuat arus listrik primer (Input), Ip = 2 A

Tegangan sekunder (Output), Vs = 44 volt

Ditanya :

Kuat arus keluaran (Output) dan daya keluarannya

Penyelesaian :

Efisiensi merupakan perbandingan antara daya keluaran (output) dengan daya masukan (input).

[tex]\eta = \frac {P_{s}}{P_{p}}\\[/tex]

Persamaan Daya :

P = V.I

Daya Keluaran (Output) :

[tex]\eta = \frac {P_{s}}{P_{p}}\\\\\frac{80}{100} = \frac {P_{s}}{V_{p}.I_{p} }\\\\\frac{4}{5}= \frac {P_{s}}{220.2}\\\\P_{s} = 352\ watt[/tex]

Diperoleh daya keluaran sebesar 352 watt.

Kuat Arus Keluaran (Output) :

[tex]P_{s} = V_{s}.I_{s} \\352 = 44.I_{s} \\I_{s} = 8\ A[/tex]

Diperoleh kuat arus keluaran sebesar 8 A.

Semoga Membantu [^_^]

VS = Tegangan Sekunder (volt)

NP = Lilitan Primer  

NS = Lilitan Sekunder

IS = Arus Sekunder ( A )

IP = Arus Primer (A)

PS = Daya Sekunder (watt)

PP = Daya Primer (watt)

Berikut penjelasan dari jawaban diatas :





Diketahui :

Vp = 240 volt

Ditanyakan : Is..........?



Dijawab :

Trafo merupakan alat yang digunakan untuk mengubah tegangan listrik bolak balik menjadi lebih besar atau lebih kecil.

Trafo sendiri dibagi menjadi 2 yaitu trafo step up (menaikan tegangan) dan trafo step down (menurunkan tegangan). Adapun ciri ciri trafo step up yaitu nilai Vs > Vp (tegangan sekunder lebih besar dibandingkan tegangan primer) , Ns > Np (jumlah lilitan sekunder lebih besar dibandingkan jumlah lilitan primer), Is < Ip (kuat arus sekunder lebih kecil dibandingkan dengan kuat arus primer). Untuk ciri ciri trafo step down yaitu nilai Vs < Vp (tegangan sekunder lebih kecil dibandingkan tegangan primer) , Ns < Np (jumlah lilitan sekunder lebih kecil dibandingkan jumlah lilitan primer), Is > Ip (kuat arus sekunder lebih besar dibandingkan dengan kuat arus primer).



Untuk mencari rumus pada transformator adalah sebagai berikut :

Np/Ns = Vp / Vs = Is/Ip

Ket:

Ns = jumlah lilitan pada kumparan primer

Np = jumlah lilitan pada kumparan sekunder

Vp = Tegangan pada kumparan primer (masukkan)

Vs = Tegangan pada kumparan sekunder (keluaran)

Ip = Arus pada kumparan primer

Is = Arus pada kumparan sekunder



Untuk mencari kuat arus pada kumparan sekunder  rumusnya sebagai berikut :

η = (Is .Vs / Ip . Vp) x 100%

80%  = (Is . 80 volt / 5 A . 240 volt) x 100%

80% = (Is x 80 volt / 1200) x 100%

80 x 1200 = Is x 8000

Is = 80 x 1200 / 8000

Is = 96.000 / 8.000

Is = 12 A

Jadi besarnya kuat arus pada kumparan sekunder adalah 12 A

32. Sebuah transformator memiliki tegangan primer 220 Volt. Transformator tersebut menghasilkan tegangan sekunder 8 Volt dan memiliki efisiensi 80 %. Jika kuat arus sekundernya 2 A, kuat arus primernya adalah


Kuat arus primernya adalah [tex]\frac{1}{11}[/tex] Ampere atau 0,091 Ampere.

Pembahasan

EFISIENSI TRANSFORMATOR TAK IDEAL

Transformator atau trafo adalah alat untuk menaikkan atau menurunkan tegangan.

Persamaan umum trafo:

[tex]\frac{Vp}{Vs} \:=\: \frac{Np}{Ns}[/tex]

Trafo ideal adalah trafo yang mampu memberikan seluruh daya masukan untuk menjadi daya keluaran.

Tetapi, pada kenyataannya sulit sekali menciptakan trafo ideal.

Efisiensi adalah perbandingan daya keluaran (output) dan daya masukan (input).

Rumus efisiensi trafo

η = [tex]\frac{Ps}{Pp}[/tex] × 100 %

η = [tex]\frac{Vs \times Is}{Vp \times Ip}[/tex] × 100 %

dimana

Vp = tegangan kumparan primer (V)Vs = tegangan kumparan sekunder (V)Np = jumlah lilitan kumparan primerNs = jumlah lilitan kumparan sekundeIp = kuat arus kumparan primer (A)Is = kuat arus kumparan sekunder (A)Pp = daya kumparan primer (W)Ps = daya kumparan sekunder (W)η= efisiensi (%)

Kumparan primer adalah kumparan yang dihubungkan ke sumber arus, sedangkan kumparan primer adalah kumparan yang dihubungkan dengan alat elektronik lainnya.

Diketahui:

Vp = 220 V

Vs = 8 V

Is = 2 A

η = 80 %

Ditanyakan:

Ip ?

Penjelasan:

η = [tex]\frac{Vs \times Is}{Vp \times Ip}[/tex] × 100 %

[tex]\frac{80 \%}{100 \%} \:=\: \frac{8 \times 2}{220 \times Ip}[/tex]

[tex]\frac{8}{10} \:=\: \frac{16}{220 \times Ip}[/tex]

8 × 220 × Ip = 10 × 16

1.760 × Ip = 160

Ip = [tex]\frac{160}{1.760}[/tex]

Ip = [tex]\frac{1}{11}[/tex] A

Ip = 0,091 A

Kuat arus primernya adalah [tex]\frac{1}{11}[/tex] Ampere atau 0,091 Ampere.

Pembahasan

Transformator

Dari soal kita ketahui Transformator adalah tipe Step down, Karena Tegangan  masuk (Primer) 220 Volt, dan Tegangan keluaran (sekunder) 8 Volt. Transformator ini memang digunakan bertujuan untuk menurunkan tegangan, Sehingga Lilitan pada kumparan sekunder akan lebih sedikit daripada lilitan di kumparan primer, akan tetapi arus yang mengalir pada kumparan sekunder lebih besar dari pada arus yang mengalir pada kumparan sekunder, Untuk Efisiensi Transformator dirumuskan dengan :

[tex]\boxed{\mu=\frac{Ps}{Pp}\times100\%=\frac{Vs\times{Is}}{Vp\times{Ip}}\times100\%}[/tex]

Keterangan  

VP = Tegangan Primer (volt)

VS = Tegangan Sekunder (volt)

IS = Arus Sekunder ( A )

IP = Arus Primer (A)

PS = Daya Sekunder (watt)

PP = Daya Primer (watt)

μ = Efisiensi (%)  

Pengerjaan Soal :

[tex]\boxed{diketahui}[/tex]

Vp = Tegangan Primer = 220 Volt

Vs = Tegangan Sekunder = 8 Volt

Is  = Kuat Arus Sekunder = 2 A

[tex]\boxed{Ditanyakan}[/tex]

Ip = Kuat Arus Primer = ....

[tex]\boxed{Penyelesaian}[/tex]

[tex]\boxed{\mu=\frac{Vp\times{Ip}}{Vs\times{Is}}\times100\%}[/tex][tex]\boxed{80\%=\frac{220\times{Ip}}{8\times2}\times100\%}[/tex][tex]\boxed{80\%=\frac{220\times{Ip}}{16}\times100\%}[/tex][tex]\boxed{16\times80=220\times{Ip}\times100}[/tex]1280 = 22.000IpIp = 1280 : 22.000Ip = 0,06 A

[tex]\boxed{Kesimpulan}[/tex]

Kuat Arus yang Mengalir Pada Kumparan Primer Sebesar 0,06 A
