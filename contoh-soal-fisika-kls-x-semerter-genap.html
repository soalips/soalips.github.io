6.)

sisi limas = 10 cm

T = 20 cm

L. permukaan limas

Lp = L ΔACB + 2L ΔTCA + L ΔTAB

perhatikan Δ ACB ⊥ C:

LΔ ACB = 1/2 x AC x BC

= 1/2 x 10 x 10

= 50 cm²

perhatikan Δ TCA ⊥ C = Δ TCB ⊥ C:

LΔ TCA = 1/2 x AC x CT

= 1/2 x 10 x 20

= 100 cm²

perhatikan Δ TAB:

tinggi segitiga = TC'

TA = TB maka ΔTAB = segitiga sama kaki

AB = alas

mencari tinggi segitiga = TC'

AB = √(AC² + BC²)

= √(10² + 10²)

= √(100 + 100)

= √200

= 10√2 cm

CC' = √(AC² - (1/2AB)²)

= √(10² - (1/2 x 10√2)²)

= √(100 - (5√2)²)

= √(100 - 50)

= √50 cm

= atau 5√2 cm

TC' = √(TC² + CC²)

= √(20² + (√50)²)

= √(400 + 50)

= √450

= √(225 x 2)

= 15√2 cm

LΔ TAB = 1/2 x AB x TC'

= 1/2 x 10√2 x 15√2

= 1/2 x 150 x 2

= 150 cm²
