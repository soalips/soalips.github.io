2. tolong ya . Dalam suatu kompetisi, Ito, Yanto dan Sam mengumpulkan total nilai 2018 point. Nilai yang dikumpulkan Yanto empat kali lebih banyak dari nilai yang dikumpulkan Sam. Namun nilai yang dikumpulkan Yanto 263 point di bawah nilai yang dikumpulkan Ito. Berapa point nilai yang dikumpulkan Yanto? tolong ya pakai cara nya


Yang bisa diketahui dari soal adalah:

     Nilai Ito + Nilai Yanto + Nilai Sam = 2018

     Nilai Yanto = 4 x Nilai Sam   ⇒  Nilai Sam = Nilai Yanto : 4

     Nilai Yanto = Nilai Ito - 263   ⇒  Nilai Ito = Nilai Yanto + 263



Pemikiran dan jawaban:


         Nilai Ito              + Nilai Yanto     +      Nilai Sam     = 2018
(Nilai Yanto + 263)      + Nilai Yanto    + (Nilai Yanto : 4) = 2018
 *** lalu ruas kanan dan ruas kiri dikalikan 4
(4 Nilai Yanto + 1052) + 4 Nilai Yanto + 4 Nilai Yanto      = 4 x 2018
4 Nilai Yanto   + 1052 + 4 Nilai Yanto + 4 Nilai Yanto       = 8072
                                              12 Nilai Yanto + 1052         = 8072
                                                                12 Nilai Yanto   = 8072 - 1052
                                                                     Nilai Yanto   = 7020 : 12

10. |x+1|+2x=7 Mohon divantu sam cara mengerjakannya yah teman" bsk mau dikumpulkan


Mapel : Matematika
Kelas : X SMA
Materi : Persamaan Mutlak

Pembahasan :
|x + 1| + 2x = 7
|x + 1| = 7 - 2x
(x + 1)² = (7 - 2x)²
(x + 1)² - (7 - 2x)² = 0
(x + 1 - 7 + 2x)(x + 1 + 7 - 2x) = 0
(3x - 6)(-x + 8) = 0
x = 2
x = 8

Hp : {2 , 8}

26. Uang Sam : uang Dicky adalah 2 : 5, sedangkan uang Sam : uang Marry adalah 4 : 7.Uang Dicky lebih banyak dari uang Sam.Perbandingan Sam: Dicky: Marry adalah?​


Uang Sam : Uang Dicky = 2 : 5

Uang Sam : Uang Marry = 4 : 7

menjadi

Uang Dicky : Uang Sam = 5 : 2

Uang Sam : Uang Marry = 4 : 7

Dicky : Sam : Marry = (5×4) : (4×2) : (2×7)

= 20 : 8 : 14

Jadi, Sam : Dicky : Marry = 8 : 20 : 14 atau 4 : 10 : 7
