Lintasan orbit bulan mengelilingi bumi hampir berbentuk lingkaran dengan jari-jari orbit Venus 1,1x10⁸ km dan periodenya 27,3 hari.tentukan kelajuan dan percepatan sentripetal Bulan mengorbit Bumi


w = 2 Phi / T = 2 x 3.14 / 27.3 = 0.23 rad / hari

v = w x r
= 0.23 x 3.85 x 10^5
v = 0.8855 x 10^5 km/hari

a_sp(percepatan sentripetal) = w^2 x r
= 0.23^2 x 3.85 x 10^5
= 0.204 x 10^5 km/hari^2
