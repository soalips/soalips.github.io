8. Jika massa proton, neutron dan inti 0/3 masing-masing adalah 1,007 sma, 1,009 sma dan 13,100 sma dimana 1 sma = 931 MeV, maka energi ikat inti atom adalah....tolong ya bingung bgt, nomor atom masa dan unsurnya ga diketahui di soal.​


Jawaban:

Massa proton = 1,007 sma = 1,007 x 931 MeV = 938,737 MeV

Massa neutron = 1,009 sma = 1,009 x 931 MeV = 940,379 MeV

Massa inti = 13,100 sma = 13,100 x 931 MeV = 12,177,100 MeV

Energi ikat inti = massa inti - (massa proton + massa neutron)

Energi ikat inti = 12,177,100 MeV - (938,737 MeV + 940,379 MeV) = 11,297,984 MeV

Jadi, energi ikat inti atom adalah 11,297,984 MeV.

28. 30. Massa sebuah inti atom 'C = 12,0000 sma, massa proton = 1,0078 sma dan massaneutron = 1,0087 sma, besarnya defek massa pada pembentukan inti "C adalah ...​


Besarnya defek massa pada pembentukan inti ¹²C₆ adalah 0,099 sma

[tex]\texttt{ }[/tex]

PEMBAHASAN

Marilah kita ingat kembali beberapa rumus pada topik Fisika Inti berikut ini:

Defek massa adalah selisih dari massa partikel pembentuk inti dengan massa inti itu sendiri.

Δm = (Z . mp + (A-Z) . mn) - (mi)

dimana :

Δm = defek massa ( sma )

Z = nomor atom

mp = massa proton ( sma )

A = nomor massa

mn = massa neutron ( sma )

mi = massa inti ( sma )

[tex]\texttt{ }[/tex]

Energi Ikat Inti adalah energi yang diperlukan untuk memisahkan partikel-partikel pembentuk inti atom.

E = Δm × ( 931,5 Mev/sma )

dimana :

E = energi ikat inti ( Mev )

Δm = defek massa ( sma )

Mari kita sekarang mencoba untuk menyelesaikan soalnya.

[tex]\texttt{ }[/tex]

Diketahui :

nomor massa atom karbon = A = 12

nomor atom karbon = Z = 6

massa inti atom karbon = mi =12,0000 sma

massa neutron = mn = 1,0087 sma

massa proton = mp = 1,0078 sma

Ditanyakan :

defek massa = Δm = ?

Penyelesaian :

[tex]\Delta m = ( Z . m_p + (A - Z) . m_n ) - m_i[/tex]

[tex]\Delta m = ( 6 \times 1,0078 + (12 - 6) \times 1,0087 ) - 12,0000[/tex]

[tex]\Delta m = ( 6 \times 1,0078 + 6 \times 1,0087 ) - 12,0000[/tex]

[tex]\Delta m = 12,099 - 12,0000[/tex]

[tex]\boxed{\Delta m = 0,099 \texttt{ sma}}[/tex]

32. soal fisika SMA,,,,,​


Konsep Analisa Mesh

Mesh 1 :

[tex]V_1+V_2-E_1=0\\\\ V_1+V_2=E_1\\\\ I_IR_1+(I_I+I_{II})R_2=E_1\\\\ 2I_I+4(I_I+I_{II})=6\\\\ 2I_I+4I_I+4I_{II}=6\\\\ 6I_I+4I_{II}=6\\\\ 3I_I+2I_{II}=3............(1)[/tex]

Mesh 2 :

[tex]V_3+V_2-E_2=0\\\\ V_3+V_2=E_2\\\\ I_{II}R_3+(I_I+I_{II})R_2=E_2\\\\ 2I_{II}+4(I_I+I_{II})=6\\\\ 2I_{II}+4I_I+4I_{II}=6\\\\ 4I_I+6I_{II}=6\\\\ 2I_I+3I_{II}=3...................(2)[/tex]

Eliminasi persamaan (1) dan (2)

[tex]3I_I+2I_{II}=3\\ 2I_I+3I_{II}=3[/tex]

persamaan (1) dikali 2,   persamaan (2) dikali 3

[tex]6I_I-4I_{II}=6\\ 6I_I-9I_{II}=9[/tex]

                        -

[tex]5I_{II}=-3\\\\ I_{II}=-\frac{3}{5}\,A=-0,6\,A[/tex]

Substitusi [tex]I_{II}[/tex] ke persamaan (2)

[tex]2I_I+3I_{II}=3\\\\ 2I_I+3\times(-\frac{3}{5})=3\\\\ 2I_I-\frac{9}{5}=3\\\\ 2I_I=3+\frac{9}{5}=\frac{24}{5}\\\\ I_I=\frac{12}{5}=2,4\,A[/tex]

Jadi, kuat arus masing-masing adalah :

[tex]I_1=I_I=2,4\,A\\ I_2=I_{II}=-0,6\,A\\ I_3=I_I+I_{II}=2,4\,A-0,6\,A=1,8\,A[/tex]

36. Massa inti atom triton ( (_1 H 3) adalah 3,0160 sma, sedangkan massa proton 1,0078 sma dan massa metron 1,0087 sma. Jika 1 sma setara dengan 931 MeV maka energi ikat inti atom triton adalah ok​


Jawaban:

8,5652 MeV

Penjelasan:

Diketahui:

A=3

Z=1

Mi = 3,0160 sma

Mp = 1,0078 sma

Mn = 1,0087 sma

1 sma = 931 MeV

Ditanya: Energi ikat (E) = ?

Jwb

E=∆m x 931

= ((Z x Mp + (A-Z) x Mn) - Mi) x 931

= ((1 x 1,0078 + (3-1) x 1,0087) - 3,0160) x 931

= ((1,0078 + 2 x 1,0087) - 3,0160) x 931

= (3, 0252 - 3,0160) x 931

= 0,0092 sma x 931 = 8,5652 MeV

PEMBAHASAN

>> Defek Massa dan Energi Ikat Inti

Inti atom tersusun dari proton dan neutron sehingga massa inti seharusnya sama dengan jumlah total massa nukleon - nukleonnya. Namun, pada kenyataannya massa inti selalu lebih kecil daripada total massa nukleon - nukleonnya. Selisih massa antara keduanya disebut defek massa (∆m). Menurut kesetaraan massa dan energi yang dikemukakan Einstein, defek massa ini berubah menjadi energi yang terdapat dalam inti atom. Energi ini disebut Energi Ikat inti. Berikut hukum kesetaraan antara massa dan energi :

[tex] \boxed{\boxed{\bold{\Delta E = \Delta m \times {c}^{2}}}}[/tex]

Keterangan :

∆E = Energi ikat inti (J) ∆m = defek massa (kg) c = 3 × 10⁸ m/s

Untuk ∆m dalam sma di peroleh :

[tex] \boxed{\boxed{\bold{\Delta E = \Delta m \times (931 \:\: MeV / sma) }}}[/tex]

Jika diketahui nomor massa dan nomor atom, maka nilai energi ikat inti :

[tex] \boxed{\boxed{\bold{\Delta E = [Z \times m_p + (A - Z) m_n - m_{inti}] \times 931 \:\: MeV}}}[/tex]

Keterangan :

Z = proton (A - Z) = neutron mp = massa proton mn = massa neutron m inti = massa inti
