10. sebutkan langkah-langkah start panjang,siap,ya!​


1.Langkahkan kaki kanan di depan dan posisikan kaki kiri di belakang.2.Turunkann letakkan kedua tangan berada di belakang garis start. 3.Letakkan lutut pada kaki belakang berada di posisi yang segaris dengan tumit di kaki depan atau bisa mundur lagi.

4.Bentuk huruf V dengan ibu jari dan keempat jari lainnya berdempetan.

5.Posisikan kedua tangan sejajar dengan garis start.

6.Luruskan kedua lengan dan fokuskan pandangan ke depan.

7.Angkat panggul hingga posisi pantat lebih tinggi dibandingkan pundak.

8.Tetap luruskan lengan dan lemaskan leher.

9.Lalu tolak balok sekuat mungkin dengan kaki belakang dan mulai berlari.

12. Tugas jelaskan cara melakukan start pendek ,(bunch start) start menengah (medium start ) dan start panjang (long start ) ​


Berikut adalah cara pelaksanaan dari short start, medium start dan long start untuk cabang lari jarak pendek.

Short Start :

Kaki kanan di depan, kaki kiri di belakang. Jari-jari kaki belakang segaris dengan tumit kaki depan

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

Medium Start :

Kaki kanan di depan, kaki kiri di belakang. Lutut segaris dengan ujung jari

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

Long Start :

Kaki kanan di depan, kaki kiri di belakang. Lutut segaris dengan tumit

Kedua tangan di garis start, turunkan badan.

Kedua lengan diluruskan lalu pandangan depan.

Posisi pantat lebih tinggi dibanding bahu

Lemahkan leher sambil lengan diluruskan

Tolak balok sekuat tenaga, lalu lari.

Jawaban:

1. start  pendek

 dilakukan dengan cara membuka kaki terbuka selebar bahu dan meletakkan kaki depan pada pijakan papan start depan dan kaki lainnya diletakkan pada pijakan papan start belakang. Posisi lutut kaki belakang diletakkan sejajar dengan jari-jari kaki depan atau bahkan lebih maju dari kaki depan.

2. start menengah                                                                      
 dilakukan dengan cara meletakkan tangan lebih lebar dari bahu. Kemudian letakkan salah satu kaki di depan dengan jarak 40 cm dari garis start. Sejajarkan lutut kaki bagian belakang dengan ujung kaki yang berada di depan.

3. start panjang
  dilakukan dengan cara  Lutut kaki belakang berada di samping tumit kaki depan. Kaki kiri berada di depan dengan posisi jinjit. Letak ujung kaki belakang segaris dengan tumit kaki depan adalah sikap start jongkok panjang.

35. Penjelasan start pendek, start menengah, start panjang



-start pendek /bunch start yaitu lutut kaki belakang diletakkan didepan ujung jari kaki depan,dengan jarak antara kedua kaki satu kepal
-start menengah/medium start yaitu lutut kaki belakang diletakkan segaris dengan ujung jari kaki depan,dengan jarak antara kedua kaki satu kepal
-start panjang/long start yaitu lutut kaki belakang diletakkan sejajar/sedikit di belakang tumit kaki depan m,dengan jaeak antara kedua kaki satu kepal
