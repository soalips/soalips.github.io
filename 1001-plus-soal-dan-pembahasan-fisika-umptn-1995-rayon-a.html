2. Soal no 510 Umptn/b/1995


f(x) = x^3 + 3x^2 + 5 akan turun jika f'(x) < 0
f'(x) = 3x^2 + 6x < 0
=> 3x(x + 2) < 0
x = 0 atau x = -2
+++ (-2) --- (0) +++
-2 < x < 0Kelas : XI (2 SMA)
Materi : Turunan
Kata Kunci : fungsi turun, penyelesaian

Pembahasan :
Diketahui fungsi kuadrat f(x) = x³ + 3x² + 5
Agar fungsi f(x) merupakan fungsi turun, maka f'(x) < 0.
Sehingga
3x² + 6x < 0

Untuk menentukan penyelesaian dari pertidaksamaan kuadrat, maka kita ubah dahulu menjadi persamaan kuadrat
3x² + 6x = 0
⇔ x² + 2x = 0
⇔ x(x + 2) = 0
⇔ x = 0 V x + 2 = 0
⇔ x = 0 V x = -2
Perhatikan gambar terlampir.
Daerah yang negatif merupakan daerah fungsi yang turun, yaitu
-2 < x < 0.

Semangat Belajar!

saat di simpangan maksimum, Ek = 0

a = -ω² A sin θ

Nah, sesuai peraturan, jika konsentrasi ditambah, maka kesetimbangan akan bergeser ke sebelah, artinya kesetimbangan pada soal akan bergeser dan membuat jumlah P2Q bertambah, dan jumlah unsur P dan Q berkurang. Sehingga obsein D paling tepat:

*Unsur P berkurang dari 4 menjadi 3
*Unsur Q berkurang dari 2 menjadi 1
*Senyawa P2Q bertambah dari 2 menjadi 4
