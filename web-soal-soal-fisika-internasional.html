Terdiri dari 2 (dua) macam, yaitu pemrograman web server dengan permograman web client

Pemrograman web server seringkali berhubungan dengan basis data (database), bahasa-bahasa pemrograman web server sangat beragam, mulai dari PHP (Personal Home Page Hypertext Preprocessor), JSP (Java Server Page), ASP (Active Server Pages) dan banyak lagi.

18. [Soal Fisika]suatu alat listrik memiliki spesifikasi 350 w/220 v. jika alat tersebut digunakan selama 20 menit, besar energi yang terpakai adalah......(Menggunakan Cara and no copas lewat web)​


Jawaban:

P = 350 W

V = 220 V

T = 20 menit = 1.200 S

Maka:

W = pt

W = (350)(1.200)

= 420.000 j

W = 420 kj

jadi, besar energi yang terpakai adalah 420 kj

suatu alat listrik memiliki spesifikasi 350 w/220 v. jika alat tersebut digunakan selama 20 menit, besar energi yang terpakai adalah

Penyelesaian :

W = V × T

W = 220 × ( 20 × 60 )

W = 220 × 1.200

W = 264.000 Joule

W = 264.000/1000

W = 264 Kj

Kesimpulan :

Jadi, besar energi yang terpakai adalah 264 Kj.

1. Mozilla Firefox

kekurangannya diantaranya:

- Startup time (waktu menjalankan program pertama kali) sangat lambat dan berat.

- Statis, fiturnya kurang banyak
