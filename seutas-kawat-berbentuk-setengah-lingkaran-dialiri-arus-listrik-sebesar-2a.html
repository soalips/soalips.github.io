12. Seutas kawat yang panjangnya 2 meter dialiri arus listrik sebesar 2A. Berapa besar Medan magnet dan Gaya Lorenzt di titik A pada jarak 2 cm dari kawat tersebut?​


Penjelasan dengan langkah-langkah:

L = 2 m

a = 2 cm = 2 x 10-² m

I = 2 A

Mo = 4π x 10^-7

F = (Mo x I)/(2π x a)

F = (4π x 10^-7 x 2)/(2π x 2 x 10-²)

F = 8π/4π x 10^(-7-(-2))

F = 2 x 10^-5 N

B = F/(I x L)

B = 2 x 10^-5/(2 x 2)

B = ½ x 10^-5

B = 5 x 10^-6 Tesla

B =  (20 π.10⁻⁷)/0,2

36. Seutas kawat dibentuk menjadi lingkaran yang berjari-jari 10π cm, kemudian dialiri arus listrik sebesar 5 A. Besar induksi magnetik di titik pusat lingkaran adalah ….​


Untuk menentukan besar induksi magnetik pada suatu titik, kita dapat menggunakan Hukum Faraday-Lenz. Menurut Hukum Faraday-Lenz, besar induksi magnetik pada suatu titik adalah sama dengan konstanta magnetis kali luas lingkaran dibagi jarak dari titik tersebut ke pusat lingkaran. Jadi, besar induksi magnetik pada titik pusat lingkaran adalah sama dengan konstanta magnetis kali 5π cm^2 dibagi 10π cm = 5 * 10^-7 * 5π / 10π = 0,00005 T. Jadi, besar induksi magnetik pada titik pusat lingkaran adalah 0,00005 T.
