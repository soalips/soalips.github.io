4. cosinus sudut antara vektor a = -i + j dan vektor B = -i dikurang 2 J + 2K adalah​


Penjelasan dengan langkah-langkah:

Untuk menghitung cosinus sudut antara dua vektor, kita dapat menggunakan rumus:

cos θ = (a . b) / (||a|| . ||b||)

dimana a dan b adalah vektor-vektor tersebut, dan θ adalah sudut antara kedua vektor tersebut. Selanjutnya, "." adalah operator dot product, dan "|| ||" adalah operator magnitude (atau panjang) dari sebuah vektor.

Pertama-tama, mari kita hitung dot product antara vektor a dan b. Dot product dari dua vektor dapat dihitung dengan mengalikan komponen-komponen vektor tersebut secara individu, kemudian menjumlahkan hasilnya. Dalam hal ini, kita memiliki:

(a . b) = (-1 * -1) + (1 * -2) + (0 * 2) = 1 + -2 + 0 = -1

Selanjutnya, mari kita hitung magnitude dari masing-masing vektor. Magnitude dari sebuah vektor dapat dihitung dengan menggunakan rumus:

||v|| = √(v1^2 + v2^2 + ... + vn^2)

dimana v1, v2, ..., vn adalah komponen-komponen dari vektor tersebut. Dalam kasus ini, kita memiliki:

||a|| = √((-1)^2 + (1)^2 + (0)^2) = √(1 + 1 + 0) = √2

||b|| = √((-1)^2 + (-2)^2 + (2)^2) = √(1 + 4 + 4) = √9 = 3

Setelah kita menghitung dot product dan magnitude dari kedua vektor tersebut, kita dapat menggunakan rumus cosinus untuk menghitung sudut antara kedua vektor tersebut. Dengan menggunakan nilai-nilai yang sudah kita hitung sebelumnya, kita memiliki:

cos θ = (-1) / (√2 * 3) = -1/6

Nilai cos θ ini menunjukkan bahwa sudut antara kedua vektor tersebut adalah sudut tegak lurus (90 derajat).

Jadi, jawabannya adalah -1/6.

15. Dua buah vektor memiliki besar yang sama.Jika hasil bagi selisih dan resultan antar kedua vektor tersebut adalah 1/2 √2 maka cosinus sudut apit antara kedua vektor tersebut adalah......


selisih vektor: √(f1² + f2² - 2.f1.f2.cos α)
resultan vektor: √(f1² + f2² + 2.f1.f2.cos α)

√(f1² + f2² - 2.f1.f2.cos α) / √(f1² + f2² + 2.f1.f2.cos α) = 1/2√2

**kuadratkan**(f1² + f2² - 2.f1.f2.cos α) / (f1² + f2² + 2.f1.f2.cos α) = 1/4(2)
(f1² + f2² - 2.f1.f2.cos α) / (f1² + f2² + 2.f1.f2.cos α) = 1/2

**ingat f1=f2**
(2f² - 2.f².cos α) / (2f² + 2.f².cos α) = 1/2
(2f² - 2.f².cos α) = 1/2 (2f² + 2.f².cos α)
2f² - 2.f².cos α = f² + f².cos α
2f² - f² = 2.f².cos α + f².cos α
f² = 3.f².cos α

**coret f²**
1 = 3.cos α
cos α = 1/3
α = 70,5°

vektor b = ( 3 , -4 )

cos α = [ (vektor a) . (vektor b) ] ÷ [ |vektor a| x |vektor b| ]

(vektor a) . (vektor b) = (-12).(3) + (5).(-4) = -36 + (-20) = -56

|vektor b| = √( 3² + [-4]² ) = √(9 + 16) = √25 = 5

Sehingga :

cos α = [ (vektor a) . (vektor b) ] ÷ [ |vektor a| x |vektor b| ]

cos α = [ -56 ] ÷ [ 13 x 5 ]

cos α = -56 / 65

33. nilai cosinus sudut antara vektor a=1,5 dan b=0,2​


Jawaban:

0.980

Penjelasan dengan langkah-langkah:

Untuk menghitung nilai kosinus sudut antara dua vektor, kita perlu menggunakan rumus:

cos θ = (a · b) / (|a| |b|)

Di mana a · b adalah hasil perkalian titik antara vektor a dan b, |a| adalah magnitudo (panjang) vektor a, dan |b| adalah magnitudo (panjang) vektor b.

Dalam kasus ini, vektor a = (1, 5) dan vektor b = (0, 2).

Untuk menghitung perkalian titik antara kedua vektor, kita akan mengalikan komponen-komponennya dan menjumlahkannya:

a · b = (1 * 0) + (5 * 2) = 0 + 10 = 10

Untuk menghitung magnitudo (panjang) vektor a, kita menggunakan rumus:

|a| = √(a₁² + a₂²)

Di mana a₁ dan a₂ adalah komponen-komponen vektor a.

|a| = √(1² + 5²) = √(1 + 25) = √26

Untuk menghitung magnitudo (panjang) vektor b, kita menggunakan rumus yang sama:

|b| = √(b₁² + b₂²)

Di mana b₁ dan b₂ adalah komponen-komponen vektor b.

|b| = √(0² + 2²) = √(0 + 4) = √4 = 2

Sekarang kita dapat menggabungkan semua nilai ini ke dalam rumus kosinus sudut:

cos θ = (a · b) / (|a| |b|)

= 10 / (√26 * 2)

Untuk mencari nilai akhir, kita dapat mengevaluasi rumus ini:

cos θ ≈ 10 / (2 * 5.099)

= 10 / 10.198

≈ 0.980

Jadi, nilai cosinus sudut antara vektor a = (1, 5) dan vektor b = (0, 2) adalah sekitar 0.980.


34. Diketahui vektor A(1,-2,1) B(-1,0,-1) dan C(-2,1,3) jika cosinus sudyt antara AB dan BC tentukanlah cosinusnya!


Jawaban:

-2 / (3√6)

Penjelasan dengan langkah-langkah:

Untuk mencari cosinus sudut antara dua vektor, kita dapat menggunakan rumus:

cos θ = (A . B) / (|A| |B|)

Di sini, A . B adalah hasil perkalian dot antara vektor A dan vektor B, |A| adalah magnitude (panjang) dari vektor A, dan |B| adalah magnitude (panjang) dari vektor B.

Mari kita hitung:

A = (1, -2, 1)

B = (-1, 0, -1)

C = (-2, 1, 3)

AB = B - A = (-1, 0, -1) - (1, -2, 1) = (-2, 2, -2)

BC = C - B = (-2, 1, 3) - (-1, 0, -1) = (-1, 1, 4)

|AB| = √((-2)^2 + 2^2 + (-2)^2) = √(4 + 4 + 4) = √12 = 2√3

|BC| = √((-1)^2 + 1^2 + 4^2) = √(1 + 1 + 16) = √18 = 3√2

AB . BC = (-2)(-1) + 2(1) + (-2)(4) = 2 + 2 - 8 = -4

cos θ = (-4) / (2√3 * 3√2) = -4 / (6√6) = -2 / (3√6)

Jadi, cosinus sudut antara AB dan BC adalah -2 / (3√6).
