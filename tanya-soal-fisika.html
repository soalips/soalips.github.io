9. soal fisika yg di tanya Rtotal , terimakasih​


25/4 ohm

Penjelasan:

Skema diatas termasuk rangkaian campuran

Dik : R1 = 2ohm

R2 = 3 ohm

R3 = 9ohm

R4 = 2ohm

Dit : R total..?

Penyelesaian :

Kerjakan dulu rangkaian paralel yaitu R2 dan R3

1/Rp = 1/R2 + 1/R3

1/Rp = 1/3 + 1/9

1/Rp = (3 + 1 )/9

1/Rp = 4/9

Rp = 9/4ohm

lalu kerjakan rangkaian seri

Rs = R1 + Rp + R4

Rs = 2 + 9/4 +2

Rs = 4 + 9/4

Rs =(16+9)/4

Rs = 25/4 ohm

Jadi R total adalah 25/4 ohm

Q2 = 0 C

Q2' = 5000 + 0 - 3500

C2 = 1500 / 35
