PENJELASAN:

⇔ Downloading atau pengunduhan merupakan istilah tentang teknologi informasi yang menyatakan proses transfer dari dua komputer atau server, dimana komputer pengguna melakukan proses pengambilan data elektronik dari komputer server.

⇔ Satuan ukuran data dalam komputer adalah:

Byte, dengan ekuivalen 8 bit.

Kilobyte, dengan ekuivalen 1024 byte.

Megabyte, dengan ekuivalen 1024 kilobyte.

Gigabyte, dengan ekuivalen 1024 megabyte.

Ukuran file video = 3 GB

35. Ukuran file yang akan didownload adalah 70MB (megabyte), sedangkan kecepatan internet adalah 3. 6Mbps (megabit per second). Maka untuk mengitung waktu yang butuhkan untuk mendownload file yang ukurannya adalah 70MB adalah.


Jawaban:

Untuk mengitung waktu yang dibutuhkan untuk mendownload file dengan ukuran 70MB, pertama-tama kita perlu mengkonversi kecepatan internet dari Mbps (megabit per second) ke MBps (megabyte per second). Hal ini dapat dilakukan dengan membagi kecepatan internet dalam Mbps dengan 8 (karena 1 byte adalah 8 bit). Dengan demikian, kecepatan internet dalam MBps adalah 3.6 / 8 = 0.45 MBps.

Setelah mengetahui kecepatan internet dalam MBps, kita dapat mengitung waktu yang dibutuhkan untuk mendownload file dengan ukuran 70MB dengan menggunakan rumus waktu = ukuran file / kecepatan internet. Dengan menggunakan rumus tersebut, kita dapat menghitung waktu yang dibutuhkan untuk mendownload file sebagai berikut:

waktu = 70MB / 0.45 MBps

waktu = 156.666667 detik

Jadi, waktu yang dibutuhkan untuk mendownload file dengan ukuran 70MB adalah 156.666667 detik atau sekitar 2,6 menit.

38. Sebuah jaringan memiliki Bandwith 512 kbps, akan mendownload file dengan ukuran 600 byte berapa waktu yang di butuhkan untuk mendownload file tersebut


Bandwith 512 Kbps (Kilobits/s) = 64 KBps (Kilobytes/s) = 65.536 Bytes/s

Kalau ukuran file = 600 Byte (Bentar, rasanya nggak mungkin 600 Bytes), maka lama download :

600 B : 65.536 B/s = 0,009... detik.

Kalau ukuran file = 600 Kilobyte (Ini yang peling mungkin), maka lama download :

600 KB : 64 KB/s = 9,375 detik.

40. Kina ingin mendownload video ukuran 3 Gigabyte menggunakan kecepatan akses yang stabil 10 mbps waktu yang dibutuhkan Gina untuk menyelesaikan proses download internet adalah berapa menit​


Jawaban:

Untuk menghitung waktu yang dibutuhkan Kina untuk menyelesaikan proses download, kita perlu mengkonversi kapasitas file (dalam hal ini, 3 Gigabyte) ke megabit (Mbps), mengingat kecepatan akses internetnya adalah 10 Mbps. Kemudian kita dapat menggunakan rumus sederhana berikut:

Waktu (dalam menit)= Kecepatan (dalam Mbps)/Kapasitas (dalam Megabit)

1 Gigabyte (GB) = 8.000 Megabit (Mb) (karena 1 byte = 8 bit)

Jadi, kapasitas file 3 GB adalah setara dengan:

3GB×8.000Mb/GB=24.000Mb

Sekarang kita dapat menghitung waktu yang dibutuhkan:

Waktu (dalam menit)=24.000Mb/10Mbps=2.400menit

Untuk mengonversi waktu dari menit ke jam, kita tahu bahwa 1 jam sama dengan 60 menit, jadi:

2.400 menit =2.400/60 jam = 40 jam

Jadi, Kina akan membutuhkan sekitar 40 jam untuk menyelesaikan proses download video berukuran 3 Gigabyte dengan kecepatan akses internet 10 Mbps.
