1. apa contoh dari energj panas bumi​


Jawaban:

Contoh energi panas bumi :

 -  Energi uap basah : dapat menggerakkan turbin generator listrik

 -  Energi panas bumi air panas : dapat menggerakkan turbin

  - Energi panas bumi batuan panas : dapat menggerakan turbin

Pembahasan  

Energi panas bumi merupakan sumber energi yang ramah lingkungan sebab berasal dari panas dalam bumi. Air yang terpompa ke dalam bumi oleh manusia dikumpulkan ke permukaan bumi dalam bentuk uap yang dimana dapat digunakan dalam penggerakan turbin-turbin untuk memproduksi listrik. Indonesia adalah saah satu negara berkembang yang dapat menghadapai peningkatan listrik sebanyak 10% setiap tahunnya. Pemerintah Indonesia telah melaksanakan upaya untuk membuat investasi energi panas bumi lebih menarik. Beberapa faktor yang menjadi penghalang adalah tarif listrik yang tidak kompetitif.

1. Diketahui Dc = c, Dc - c, Dc - H, Dc - Br, dan DH - Br berturut turut : 614 kj/mol, 348 kj/mol, 413 kj/mol, 276 kj/mol, dan 366 kj/mol. Maka perubahan entalpi pada adisi 5,6 gram etena oleh HBr adalah -11,4 kJ/mol.

2. Diketahui energi ikatan :

C = C = 614 kJ/mol

C - O = 358 kJ/mol

C - C = 348 kJ/mol

C - H = 414 kJ/mol

H - O = 463 kJ/mol

nilai ΔH dari reaksi : CH₂ = CH - CH₃ + H₂O ⇒ CH₃ - CH₂ - CH₂ - OH adalah -43 kJ/mol.

Penyelesaian Soal :

1. Diketahui : Dc = c = 614 kj/mol

                  Dc - c = 348 kj/mol

                  Dc - H = 413 kj/mol

                  Dc - Br = 276 kj/mol

                  DH - Br = 366 Kj/mol

                  Massa etena = 5,6 gram

Ditanya : ΔH ?

Jawab :

LANGKAH PERTAMA (I)

Reaksi adisi etena :

CH₂ = CH₂ + HBr ⇒ CH₃ - CH₂ - Br

H   H                                H   Br

Ι     Ι                                 Ι     Ι

C = C  +  H - Br   ⇒   H - C - C - H

Ι     Ι                                 Ι     Ι

H    H                               H   H

LANGKAH KEDUA (II)

Hitung ΔHr dengan menggunakan cara sebagai berikut :

ΔHr = ∑D Pereaksi - ∑D hasil reaksi

      = {Dc = c + (4 × Dc - H) + DH-Br} - {Dc-c + (5 × Dc-H) + DC-Br}

      = {614 + (4 × 413) + 366} - {348 + (5 × 413) + 276}

      = {614 + 1652 + 366} - {348 + 2065 + 276}

      = 2632 kJ/mol - 2689 kJ/mol

      = -57 Kj/mol

LANGKAH KETIGA (III)

Hitung Massa molekul relatif (Mr) etena (C₂H₄) dengan menggunakan cara sebagai berikut :

Mr C₂H₄ = (2 × Ar C) + (4 × Ar H)

              = (2 × 12 g/mo) + (4 × 1 g/mol)

              = 24 g/mol + 4 g/mol

              = 28 g/mol

LANGKAH KEEMPAT (IV)

Hitung mol etena (C₂H₄) dengan menggunakan cara sebagai berikut :

mol = massa/ Mr

      = 5,6 gram/ 28 g/mol

      = 0,2 mol

LANGKAH KELIMA (V)

Hitung Perubahan entalpi adisi etena dengan cara sebagai berikut :

ΔH = ΔHr × mol

     = -57 kJ/mol × 0,2 mol

     = -11,4 Kj/mol

∴ Kesimpulan nilai perubahan entalpi adalah -11,4 kJ/mol.

2. Diketahui energi ikatan :

C = C = 614 kJ/mol

C - O = 358 kJ/mol

C - C = 348 kJ/mol

C - H = 414 kJ/mol

H - O = 463 kJ/mol

reaksi : CH₂ = CH - CH₃ + H₂O ⇒ CH₃ - CH₂ - CH₂ - OH

Ditanya : nilai ΔH ?

Jawab :

ΔHr = ΣEI zat pereaksi – ΣEI zat hasil reaksi

       = {C = C + C - C + (6 × C - H) + (2 × H - O)} - {(2 × C - C) + C - O + (7 × C - H) + H - O}

       = {614 + 348 + (6 × 414) + (2 × 463)} - {(2 × 348) + 358 + (7 × 414) + 463}

       = {614 + 348 + 2484 + 926} - {696 + 358 + 2898 + 463}

       = 4372 kJ/mol - 4415 kJ/mol

       = -43 kJ/mol

∴ Kesimpulan nilai ΔH reaksi tersebut adalah -43 kJ/mol.

31. sebuah bola besi massanya 0,2 kg dilempar vartikel ke atas. Energj potensial benda pada ketinggian maksimum adalah 40 J. bila g = 10 m/s2, maka hitunglah ketinggian maksimum yang dicapai bola tersebut

Energi Potensial

dik :
Ep (Energi Potensial) = 40 J
m (massa) = 0,2 kg
g (percepatan gravitasi) = 10 m/s²

dit :
h (ketinggian)?

jawab :

Ep = m × g × h

h = Ep / m × g
h = 40 / 0,2 × 10
h = 40 / 2
h = 20 m
