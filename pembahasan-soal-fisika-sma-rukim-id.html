9. perbedaan net id dan host id


Net ID berperan dalam identifikasi suatu network dari network yang lain, sedangkan host ID berperan untuk identifikasi host dalam suatu network. Jadi, seluruh host yang tersambung dalam jaringan yang sama memiliki net ID yang sama. Garis pemisah antara bagian network dan host tidak tetap, bergantung kepada kelas network. IP address dibagi ke dalam lima kelas, yaitu kelas A, kelas B, kelas C, kelas D dan kelas E.

12. Pengertian network id, host id, dan broadcast id


Pengertian✍️

Network id

Network ID adalah bagian dari IP address yang menunjukkan di jaringan mana komputer tersebut berada.

Host id

Host adalah identitas dari host (interface router, workstation, dan device) yang terhubung ke jaringan.

Broadcast id

broadcast adalah alamat jaringan yang digunakan untuk mengirim ke semua perangkat yang terhubung ke jaringan komunikasi akses ganda.
