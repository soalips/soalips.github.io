1. Jika ayunan sederhana bergetar sebanyak 50 kali dalam waktu 10 sekon. Maka besar frekuensi ayunan dan periode ayunannya adalah...


Jika ayunan sederhana bergetar sebanyak 50 kali dalam waktu 10 sekon , maka besar frekuensi ayunan dan periode ayunannya adalah 5 Hz dan 0,2 sekon

PEMBAHASAN

Gelombang sinusoidal adalah gelombang yang membentuk fungsi sinus. Beberapa rumus dasar gelombang yang perlu di ingat adalah sebagai berikut ini :

f = 1 / T

v = λ . f

Δx / λ = ΔΘ / 2π

Δt / T = ΔΘ / 2π

dimana :

f = frekuensi gelombang ( Hz )

T = periode gelombang ( s )

v = cepat rambat gelombang ( m/s )

λ = panjang gelombang ( m )

Δx = jarak pisah antar dua titik pada gelombang ( m )

Δt = selang waktu ( s )

ΔΘ = beda fase ( radian )

Bentuk persaman gelombang adalah :

y = A sin ( ωt - kx )

dimana :

y = simpangan gelombang ( m )

A = amplitudo gelombang ( m )

ω = frekuensi sudut ( rad/s )

t = waktu ( s )

k = bilangan gelombang ( 1/m ) = 2π/λ

x = jarak titik dari sumber ( m )

Okay mari kita gunakan rumus ini untuk menyelesaikan soalnya.

Diketahui :

Jumlah getaran = n = 50 kali

Waktu getaran = t = 10 sekon

Ditanyakan :

f = ?

T = ?

Penyelesaian :

f = n / t

f = 50 / 10

f = 5 Hz

T = t / n

T = 10 / 50

T = 0,2 sekon

29. Bagaimana jalan untuk mendapatkan hasil frekuensi ayunan dan priode ayunan soal: jika ayunan sederhana bergetar sebanyak 60 kali dalam waktu 15 sekon.Tentukan


a. Frekuensi getaran adalah [tex]\boxed{\bold{4 \: Hz}}[/tex]

b. Periode getaran adalah [tex]\boxed{\bold{0,25 \: s}}[/tex]

Pembahasan

Hi! soal tersebut adalah membahas mengenai getaran dan gelombang. Singkatnya, getaran adalah pergerakan berbolak balik di sekitar keseimbangan. Biasanya getaran di ilustrasikan dengan bandul atau gerakan pegas.

Gerakan bandul :

Jika bandul A-B = [tex]\bold{\frac{1}{4}}[/tex] getaran. Jika bandul A-B-C = [tex]\bold{\frac{1}{2}}[/tex] getaran. Jika bandul A-B-C-B = [tex]\bold{\frac{3}{4}}[/tex] getaran. Jika bandul A-B-C-B-A = [tex]\bold{1}[/tex] getaran.

Rumus menentukan frekuensi :

[tex]\boxed{\bold{f = \frac{n}{t}}}[/tex]

Dimana :

[tex]\bold{f}[/tex] = frekuensi.

[tex]\bold{n}[/tex] = jumlah getaran dalam waktu t sekon.

Berarti :

[tex]\boxed{\bold{f = \frac{1}{T}}}[/tex]

Rumus menentukan periode :

[tex]\boxed{\bold{T = \frac{t}{n}}}[/tex]

Dimana :

[tex]\bold{T}[/tex] = periode.

[tex]\bold{n}[/tex] = banyak getaran.

[tex]\bold{t}[/tex] = waktu.

Berarti :

[tex]\boxed{\bold{T= \frac{1}{f}}}[/tex]

Langkah Penyelesaian

Diketahui :

Banyak getaran = 60 kali Waktu = 15 sekon.

Ditanya :

a. Frekuensi getaran? b. Periode getaran?

Dijawab :

Mencari frekuensi :

[tex]\bold{f = \frac{n}{t}}[/tex]

[tex]\bold{f = \frac{60}{15}}[/tex]

[tex]\bold{f = 4 \: Hz}[/tex]

Mencari periode :

[tex]\bold{T = \frac{t}{n}}[/tex]

Dit : periode ayunan

T = t / n

37. jawaban lks ipa kelas 8 semester 2 evaluasi 10 1-45sebuah ayunan sederhana bergetar dengan frekuensi 5 Hz berapa kali ayunan tersebut jika berayun selama 1 menit ​


 Bandul merupakan benda yang terikat tali dan dapat berayun. Benda berayun melalui titik kesetimbangan dan kembali lagi keposisi awal.

Pembahasan  

Berdasarkan soal diatas diketahui bahwa:

sebuah ayunan sederhana bergetar dengan frekuensi 5 Hz dan berayun selama 1 menit -> f = 5 Hz, t = 1 menit = 60 detik

Adapun yang ditanyakan pada soal diatas adalah menentukanberapa kali ayunan tersebut (n).

Frekuensi merupakan banyaknya getaran tiap satu menit.  

Persamaan atau rumus mencari frekuensi pada bandul matematis yaitu sebagai berikut:

f = n/t  

keterangan:

f : frekuensi (satuan Hz) n : jumlah bunyi/ gelombang t : waktu (satuan detik atau sekon)

Sedangkan periode merupakan waktu yang dibutuhkan untuk satu kali getaran.  

Adapun Persamaan atau rumus mencari periode pada bandul matematis yaitu sebagai berikut:  

T =  t/n

keterangan:

n : jumlah bunyi/ gelombang t : waktu (satuan detik atau sekon) T : periode (satuan sekon)

Nah, setelah diketahui rumusnya diatas maka untuk mencari berapa kali ayunan tersebut caranya sebagai berikut:

f = n/t

n = f x t

n = 5 x 60

n = 300 kali

Dengan demikian dapat disimpulkan bahwa banyaknya bandul berayun adalah 300 kali ayunan.
