4. fisika penjumlahan vektor tentukan resultan nya mohon di bantu​


Penjelasan:

sigma Fx = F2 sin 30° - F1 sin 60°

= (1× ½) - (2 × ½√3)

= (½ - √3) N

sigma Fy = F1 cos 60° - F2 cos 30°

= (2×½) - (1× ½√3)

= (1 - ½√3) N

Resultan vektor :

R² = (sigma Fx)² + (sigma Fy)²

= (½ - √3)²+(1 - ½√3)²

= (¼ - √3 +3) +(1 - √3 +¾)

= ¼+¾+3+1 -√3-√3

= 5 - 2√3

=> R = √(5 - 2√3)) N

6. Hitunglah resultan vektor dari soal di atas


~> Proyeksi vektor gaya pda sumbu horisontal/x :

F3x = F3.cos 60°

= 10.0,5 ~ 5 N ke arah kiri ✓

F2x = F2.cos 30°

= 8√3.0,5√3 ~ 12 N ke arah kiri ✓

~> Proyeksi vektor gaya pda sumbu vertikal/y :

F3y = F3.sin 60°

= 10.0,5√3 ~ 5√3 N ke arah atas ✓

F2y = F2.sin 30°

= 8√3.0,5 ~ 4√3 ke arah bawah ✓

~> ΣFx = F3x + F2x - F1

= 5 + 12 - 14 ~ 3 N ke arah kiri ✓

ΣFy = F3y - F2y

= 5√3 - 4√3 ~ √3 N ke arah atas ✓

Res.F = √ΣFx² + ΣFy²

= √9 + 3

= √12 ~ 2√3 N ke arah kiri atas/Barat Laut ✓✓


7. Suatu hukum fisika menyatakan bahwa percepatan suatu benda berbanding langsung dengan resultan gaya yang bekerja padanya dan berbanding terbalik dengan massa benda tersebut. dari pernyataan ini tentukan dimensi resultan gaya


Dari pernyataan di atas, di dapat persamaan :
ΣF = m . a

m = massa
satuan massa adalah kg
maka dimensi m adalah M

a = percepatan
satuan percepatan adalah m/s²
maka dimensinya adalah : L/T² = 

ΣF = m . a
ΣF = M . L/T²
ΣF = M L /T²
ΣF = M L T⁻²

Jadi, dimensi resultan gaya adalah M L T⁻²

21. Soal fisika persamaan titik A(2,5) dan B(6,8) tentukan perpindahan posisi, persamaan posisi dan resultan vektor


Persamaan vektor titik A(2,5) dan B(6,8). Maka:

Perpindahan posisi vektor adalah ∆r = 4i + 3jPersamaan posisi vektor adalah ∆r = 4i + 3jResultan vektor adalah 5 satuan.

Penjelasan dan langkah-langkah

Diketahui:

Vektor posisi di titik A dan B

r[tex]_A[/tex] = 2i +5j

r[tex]_B[/tex] = 6i + 8j

Ditanyakan:

perpindahan posisi = ?

persamaan posisi = ?

resultan vektor = ?
Jawab:

Perpindahan posisi vektor

Vektor perpindahan dari A ke B adalah ∆r yang diperoleh dari:

∆r = r[tex]_B[/tex] - r[tex]_A[/tex]

∆r = (6i + 8j) - (2i +5j)

∆r = 4i + 3j

Persamaan posisi:

∆r = 4i + 3j

Resultan vektor

|∆r| = [tex]\sqrt{4^{2} + 3^2}[/tex]

|∆r| = [tex]\sqrt{16+9}[/tex]

|∆r| = [tex]\sqrt{25}[/tex]

|∆r| = 5 satuan.

Resultan gaya pada sumbu X

∑ Fx = F₁ x + F₂ x + F₃ x + F₄ x

∑ Fx = 0 + 90 cos 40° - 30 cos 40° + 150

∑ Fx = 150 + 60 cos 40°

∑ Fx = 150 + 60 (0,77)

∑ Fx ≈ 196 N

Resultan gaya pada sumbu Y

∑ Fy = F₁ y + F₂ y + F₃ y + F₄ y

∑ Fy = 60 + 90 sin 40° - 30 sin 40° + 0

∑ Fy = 60 + 60 sin 40°

∑ Fy = 60 + 60 (0,64)

∑ Fy ≈ 98 N

R = √[(∑ Fx)² + (∑ Fy)²]

R = √[(196)² + (98)²]

R = √[48020]

R ≈ 98√5 N

tan θ = 98 / 196

θ ≈ 26,6° (dihitung dari sumbu X)
