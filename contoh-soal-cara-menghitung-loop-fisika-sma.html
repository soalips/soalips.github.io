2. kaa bantu yg bisa loop fisika​


loop kiri (misal dibuat searah jarum jam)

-E1 + I1 (R1 + R1) + I3.R2 + E2 = 0

-2 + 2.I1 + 2.I3 + 4 = 0

2.I1 + 2.I3 = -2 ====> I1 + I3 = -2 ...... (1)

loop kanan (misal dibuat berlawanan arah jarum jam)

-E3 + I2 (R1 + R1) + I3.R2 + E2 = 0

-4 + 2.I2 + 2.I3 + 4 = 0

2.I2 + 2.I3 = 0 ====> I2 = -I3  ...... (2)

di titik cabang a berlaku hukum I kirchoff :

I1 + I2 = I3   ====> I1 = I3 - I2  ...... (3)

persamaan (1) dan (3)

I3 - I2 + I3 = -2

2.I3 - I2 = -2

dari persamaan (2), I2 dimasukkan = -I3

2.I3 - (-I3) = -2

3.I3 = -2

I3 = -2/3 A. (ke atas)

hasil I3 dimasukkan ke persamaan (2)

I2 = -(-2/3) = 2/3 A (ke atas)

hasil I2 dimasukkan ke persamaan (3)

I1 = I3 - I2

I1 = -2/3 - 2/3

I1 = -4/3 A. (ke bawah)

Vab = I3.R2 + E2

Vab = -2/3.2 + 4

Vab = 8/3 Volt

Hitung pangkat sebuah nilai berdasarkan angka dan nilai pangkatnya. Buat tiga versi dari program ini menggunakan while-loop, do-while dan for-loop dalam bahasa pemrograman C++ !Jawaban & Penjelasan:

Oke, pertama-tama pastikan anda mengerti mengenai dasar operator aritmatika C++ dan sistematika perulangan. Jika sudah, mari kita lanjutkan.

While-loop dan do-while sebenarnya hanya seperti fungsi invers, do-while mempunyai logika melakukan kemudian dicek kondisinya. Sedangkan while-loop mempunyai logika mengecek dulu kondisinya lalu jika TRUE akan melakukan perintah di dalam kurawalnya.

For-loop adalah jenis loop yang menggunakan tiga argumen sebelum melakukan perintah di dalam kurawalnya, yaitu for(x;xx;xxx) di mana x sebagai angka pencacahan, xx sebagai kondisi yang harus direturn TRUE untuk melanjutkan dan xxx sebagai modifier x sebagai angka pencacahan.

Saya akan menggunakan modul cmath untuk menggunakan fungsinya berupa pow() untuk menghitung nilai pangkat. Jadi, logikanya kira-kira:

Program dimulai.include modul iostream dan cmath.Deklarasi kelas int main().Deklarasi seluruh variabel yang dibutuhkan (angka basis boleh desimal berkoma).Lakukan proses Input/Output untuk angka basis, angka pangkat awal dan angka pangkat akhir (angka pangkat integer).Lakukan perulangan while-loop, do-while dan for-loop dengan keterangan di atas.Cetak output berupa angka basis pangkat angka pangkat awal hingga angka pangkat akhir.Program selesai.

Untuk while-loop:

___

//Program dimulai

#include <iostream>

#include <cmath>

using namespace std;

int main() {

   int i, angkaPangkatDari, angkaPangkatSampai;

   float angkaBasis, hasil;

   cout << "Masukkan angka basis: ";

   cin >> angkaBasis;

   

   cout << "Masukkan angka pangkat dari: ";

   cin >> angkaPangkatDari;

   

   cout << "Masukkan angka pangkat sampai: ";

   cin >> angkaPangkatSampai;

   i = angkaPangkatDari;

   while(i <= angkaPangkatSampai) {

       hasil = pow(angkaBasis, i);

       cout << angkaBasis << "^" << i << " = " << hasil << endl;

       i++;

   }

}

//Program selesai

Untuk for-loop:

___

//Program dimulai

#include <iostream>

#include <cmath>

using namespace std;

int main() {

   int i, angkaPangkatDari, angkaPangkatSampai;

   float angkaBasis, hasil;

   cout << "Masukkan angka basis: ";

   cin >> angkaBasis;

 

   cout << "Masukkan angka pangkat dari: ";

   cin >> angkaPangkatDari;

   

   cout << "Masukkan angka pangkat sampai: ";

   cin >> angkaPangkatSampai;

   for(i = angkaPangkatDari; i <= angkaPangkatSampai; i++) {      

       hasil = pow(angkaBasis, i);

       cout << angkaBasis << "^" << i << " = " << hasil << endl;

   }

   return 0;

}

//Program selesai

Φ jika θ = 30° =____?

Rumus fluks magnetik

B = B A cos θ

1.[|] θ = 90°,

Φ = B Α cos 90

Φ = 16 • 10⁻² • 10⁻⁴ • 25 • 1

Φ = 4 • 10⁻⁴ Wb

2. [|] θ = 30°

Φ = Β Α cos θ

Φ = 16 • 10⁻² • 10⁻⁴ • 25 • ½ √3

Φ = 2 • 10⁻⁴ √3 Wb

10. apa yang dimaksud kontrol loop terbuka dan loop tertutup?


 Sistem Kontrol Loop Terbuka (Open-Loop Control System)
    Suatu sistem kontrol yang mempunyai karakteristik dimana nilai keluaran tidak memberikan pengaruh pada aksi kontrol disebut Sistem Kontrol Loop Terbuka (Open-Loop Control System). Contoh dari sistem loop terbuka adalah operasi mesin cuci. Penggilingan pakaian, pemberian sabun, dan pengeringan yang bekerja sebagai operasi mesin cuci tidak akan berubah (hanya sesuai dengan yang diinginkan seperti semula) walaupun tingkat kebersihan pakaian (sebagai keluaran sistem) kurang baik akibat adanya faktor-faktor yang kemungkinan tidak diprediksikan sebelumnya.
Sistem Kontrol Loop Tertutup (Closed-Loop Control System)
*Sistem kontrol loop tertutup adalah identik dengan sistem kontrol umpan balik, dimana nilai dari keluaran akan ikut mempengaruhi pada aksi kontrolnya.

25. bagaimana cara membuat for loop di C++​


Jawaban:

Perulangan adalah suatu proses eksekusi statemen-statemen dalam sebuah program secara terus-menerus sampai terdapat kondisi untuk menghentikannya. Operasi perulangan / looping selalu dijumpai didalam berbagai bahasa pemrograman, hal tersebut karena struktur perulangan akan sangat membantu dalam efisiensi program.

Struktur For

Struktur pengulangan / perulangan jenis for biasanya digunakan untuk melakukan perulangan yang telah diketahui banyaknya. Biasanya jenis perulangan for dianggap sebagai jenis perulangan yang paling mudah dipahami.

Untuk melakukan perulangan dengan menggunakan struktur perulangan for, kita harus memiliki sebuah variabel sebagai indeksnya. Namun perlu sekali untuk diperhatikan bahwa tipe data dari variabel yang akan digunakan sebagai indeks haruslah tipe data yang mempunyai urutan yang teratur, misalnya tipe data int (0,1,2, ... ) atau char ('a' , 'b' , 'c' , ... ).

Adapun bentuk umum dari struktur perulangan for adalah seperti yang tampak dibawah ini:

// Untuk perulangan yang sifatnya menaik (increment)

// Pastikan nialai awal < kondisi saat berjalan

for(variabel = niali_awal ; kondisi_saat_berjalan ; variable++)

{

Statemen_yang_akan_diulang;

}

// Untuk perulangan yang sifatnya menurun (decrement)

// Pastikan nialai awal > kondisi saat berjalan

for(variabel = nilai_awal ; kondisi_saat_berjalan ; variable--)

{

Statemen_yang_akan_diulang;

}

Untuk menghitung dimensi antena loop, kita memerlukan nilai panjang gelombang pada frekuensi yang diberikan.

Panjang gelombang (λ) pada frekuensi 650 MHz dapat dihitung menggunakan rumus:

λ = c / f

di mana c adalah kecepatan cahaya dan f adalah frekuensi dalam Hz.

Untuk kecepatan cahaya, kita dapat menggunakan nilai standar 299,792,458 meter per detik.

Jadi, panjang gelombang pada frekuensi 650 MHz adalah:

λ = 299,792,458 / (650 × 10^6)

= 0.4606 meter atau 46.06 cm

Setelah mengetahui nilai panjang gelombang, kita dapat menghitung dimensi antena loop yang diinginkan.

Terdapat beberapa rumus yang dapat digunakan untuk menghitung dimensi antena loop, namun untuk tujuan ini, kita akan menggunakan rumus yang paling sederhana, yaitu:

P = λ / 2

di mana P adalah lingkarannya.

Jadi, untuk frekuensi 650 MHz, lingkaran antena loop yang diinginkan adalah:

P = λ / 2

= 0.4606 / 2

= 0.2303 meter atau 23.03 cm

Jadi, antena loop yang bekerja pada frekuensi 650 MHz harus memiliki lingkaran dengan diameter sekitar 23.03 cm. Namun, perlu diingat bahwa ini hanya merupakan dimensi teoritis, dan dimensi fisik antena loop yang sebenarnya dapat sedikit berbeda tergantung pada berbagai faktor seperti bahan dan ukuran pengikat, bentuk loop, dan faktor lingkungan lainnya.
