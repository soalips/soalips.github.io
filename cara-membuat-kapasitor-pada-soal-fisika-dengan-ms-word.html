10. tolong dijawab soal fisika tentang kapasitor ini


Pada Rangkaian Capasitansi.

Ketika beberapa capasitor di rangkai secara seri.
1/Cs = 1/C + 1/C +...
1/Cs = n 1/C
Cs = C/n

Ketika n capasitor dirangkai secara paralel.
Cp = C + C +...
Cp = nC

Kemudian rangkaian seri dan paralel di rangkai secara seri

1/C = 1/Cp + 1/Cs

1/C = 1/nC + n/C
1/C = (1/nC + n^2/nC)
1/C = (1+ n^2) / nC

C = nC / (1 + n^2)

Jadi, capasitor penggantinya adalah nC/(1+n^2)

11. jelaskan cara membuat document PDF dari MS-Word​


Penjelasan:

Pilih File > Simpan Sebagai (atau tekan Command+Shift+S), ketikkan nama file di kotak teks Simpan Sebagai lalu pilih tempat Anda ingin menyimpan file.

Di dialog Simpan Sebagai, masuk ke kotak menurun File Format. Gunakan Panah Bawah untuk menelusuri tipe file, lalu pilih PDF.

Pilih tombol radio "Cocok untuk distribusi dan aksesibilitas elektronik (menggunakan layanan online Microsoft)." Tindakan ini memastikan PDF sudah ditandai.

Pilih Simpan.

Jawaban:

1. Pilih File atau buat File yang diinginkan

2. lalu buka File tersebut, nah setelah dibuka tekan Menu Bar 'File' nah disitu akan ada Sub Menu yang mengarah Vertikal, ada 'Info, New, Save, Save As, DLL'

3. Nah kamu pilih Save As karena File kamu akan disimpan dengan format yang berbeda yaitu menjadi PDF.

4. Setelah klik Save As, pilih lokasi penyimpan

5. Seteleh ditentukan lokasinya, Anda bisa merubah nama file yang diinginkan dan Merubah Format File yang biasaya (.docx), dirubah menjadi file PDF (.pdf). Dengan cara mengklik fungsi "Save as Type: "

6. Setelah dirubah menjadi format PDF. Klik 'Save' Maka file yang akan disimpan juga akan menjadi file PDF.

Catatan: File yang diawal dan File hasil PDF itu berbeda, jadi kalian tidak merubah file awal. Namun, membuat file baru dengan isi file yang awal tapi dengan format PDF.

14. cara membuat kolom koran di MS Word​


1. Buka aplikasi Ms. Word dan bukalah dokumen yang akan dibuat menjadi kolom koran, atau ketiklah teks yang diinginkan.

2. Kemudian seleksi atau blok teks yang sudah diketik, yang akan dibuat kolom koran.

3. Selanjutnya klik “Menu Page Layout” dan pilih “Colums” (di sini kita dapat memilih jumlah kolomnya, baik itu 1 kolom, 2 kolom, 3 kolom, atau 2 kolom dominan kanan atau kiri).

4. Kita juga bisa memilih “More Columns” untuk mengatur sendiri jumlah kolom yang diinginkan.

5. Jika kita memilih “More Columns” maka akan muncul kotak dialog kolom. Pada bagian Presets (untuk menentukan model kolom yang diinginkan), Number of Columns (untuk menentukan jumlah kolom yang diinginkan), Width and Spacing (untuk mengatur lebar dan jarak kolom).

6. Terakhir klik “OK”, maka akan jadi seperti pada gambar. Demikianlah langkah-langkah membuat kolom koran pada Ms. Word.

19. cara membuat title heading pada MS Word adalah..​


Jawaban:

Cara Membuat Heading di Word dengan Mudah

Membuat heading di Microsoft Word sangatlah mudah. Kalian hanya perlu mengikuti tutorial cara membuat heading di word berikut:

Buka aplikasi Microsoft Word (double click pada ikon Word di Desktop)

Lalu pilih new document atau blank document

Selanjutnya tuliskan teks “Heading 1” untuk dijadikan judul (H1)

Di bagian atas (menu bar), tekan Home -> Styles -> H1 (contoh ada pada gambar di bawah).

Selanjutnya tekan enter, kemudian tuliskan teks “Heading 2” untuk dijadikan sub judul (H2).

Tekan Home -> Styles -> H2.

Selesai.
